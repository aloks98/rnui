---
title: Toggle
description: A two-state button that can be toggled on or off.
---

import { Toggle } from '@e412/rnui-react';
import { Bold, Italic, Underline } from 'lucide-react';

The `Toggle` component is a pressable button that toggles between on and off states. It supports `default` and `outline` variants and three sizes.

## Import

```tsx
import { Toggle } from '@e412/rnui-react';
```

## Usage

<Preview code={`<Toggle aria-label="Toggle bold">
  <Bold className="h-4 w-4" />
</Toggle>`}>
  <Toggle aria-label="Toggle bold">
    <Bold className="h-4 w-4" />
  </Toggle>
</Preview>

<Preview code={`<div className="flex gap-3 items-center">
