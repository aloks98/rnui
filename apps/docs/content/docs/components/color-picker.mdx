---
title: ColorPicker
description: A color picker with visual selector, format switching, and preset swatches.
---

import { ColorPicker } from '@e412/rnui-react';

The `ColorPicker` component provides a visual interface for selecting colors. It supports hex, RGB, and HSL formats, an optional alpha channel slider, and preset color swatches for quick selection.

## Import

```tsx
import { ColorPicker } from '@e412/rnui-react';
```

## Usage

<Preview>
  <ColorPicker value="#3b82f6" />
</Preview>

```tsx
<ColorPicker value="#3b82f6" />
```

## Variants

### With Presets

Provide a `presets` array of hex colors to display quick-pick swatches.

<Preview>
  <ColorPicker
    value="#3b82f6"
    presets={[
      '#ef4444', '#f97316', '#eab308', '#22c55e',
      '#3b82f6', '#8b5cf6', '#ec4899', '#000000',
      '#6b7280', '#14b8a6', '#f43f5e', '#ffffff',
    ]}
  />
</Preview>

```tsx
<ColorPicker
  value="#3b82f6"
  presets={[
    '#ef4444', '#f97316', '#eab308', '#22c55e',
    '#3b82f6', '#8b5cf6', '#ec4899', '#000000',
    '#6b7280', '#14b8a6', '#f43f5e', '#ffffff',
  ]}
/>
```

### RGB Format

Set `defaultFormat="rgb"` to display the color value in RGB notation.

<Preview>
  <ColorPicker value="#8b5cf6" defaultFormat="rgb" />
</Preview>

```tsx
<ColorPicker value="#8b5cf6" defaultFormat="rgb" />
```

### HSL Format

Set `defaultFormat="hsl"` to display the color value in HSL notation.

<Preview>
  <ColorPicker value="#22c55e" defaultFormat="hsl" />
</Preview>

```tsx
<ColorPicker value="#22c55e" defaultFormat="hsl" />
```

### With Alpha

Enable `showAlpha` to display an alpha transparency slider.

<Preview>
  <ColorPicker value="#3b82f6" showAlpha defaultFormat="rgb" />
</Preview>

```tsx
<ColorPicker value="#3b82f6" showAlpha defaultFormat="rgb" />
```
