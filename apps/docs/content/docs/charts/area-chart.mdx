import { AreaChart } from '@e412/rnui-react';

The `AreaChart` component renders filled area charts. It supports multiple series, stacking, and smooth curves.

## Import

```tsx
import { AreaChart } from '@e412/rnui-react';
```

## Usage

<Preview code={`<div className="w-full max-w-lg">
    <AreaChart
      categories={['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']}

        { name: 'Sessions', data: [12000, 14000, 11000, 18000, 20000, 23000] },
      ]}
      height={350}
    />
  </div>`}>
  <div className="w-full max-w-lg">
    <AreaChart

        { name: 'Sessions', data: [12000, 14000, 11000, 18000, 20000, 23000] },
      ]}
      height={350}
    />
  </div>
</Preview>

## Stacked

Set `stacked` to stack areas on top of each other, useful for showing composition over time.

<Preview code={`<div className="w-full max-w-lg">

      height={350}

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `categories` | `string[]` | required | X-axis category labels |
| `series` | `{ name: string; data: number[] }[]` | required | Series data |
| `stacked` | `boolean` | `false` | Stack areas on top of each other |
| `smooth` | `boolean` | `false` | Use smooth bezier curves |
| `showLegend` | `boolean` | `false` | Show chart legend |
| `height` | `number` | `350` | Chart height in pixels |
| `loading` | `boolean` | `false` | Show loading state |
| `option` | `object` | -- | Override or extend the ECharts option |
