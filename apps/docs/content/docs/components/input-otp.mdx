---
title: Input OTP
description: A one-time password input component with individual character slots and separator support.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
  InputOTPSeparator,
} from '@e412/rnui-react'
```

## Usage

```tsx
<InputOTP maxLength={6}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
    <InputOTPSlot index={3} />
    <InputOTPSlot index={4} />
    <InputOTPSlot index={5} />
  </InputOTPGroup>
</InputOTP>
```

## Props

### InputOTP

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxLength` | `number` | - | Maximum number of characters |
| `value` | `string` | - | Controlled value |
| `onChange` | `(value: string) => void` | - | Change handler |
| `disabled` | `boolean` | `false` | Disabled state |
| `containerClassName` | `string` | - | CSS class for the container |
| `className` | `string` | - | Additional CSS class |

Accepts all props from the `input-otp` library's `OTPInput` component.

### InputOTPGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS class |

### InputOTPSlot

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `index` | `number` | - | Slot index (zero-based) |
| `className` | `string` | - | Additional CSS class |

### InputOTPSeparator

Renders a dash icon between OTP groups. Accepts standard `<div>` props.
