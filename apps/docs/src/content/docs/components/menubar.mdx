---
title: Menubar
description: A horizontal menu bar with dropdown menus, built on Base UI and composing the DropdownMenu component.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  Menubar,
  MenubarMenu,
  MenubarTrigger,
  MenubarContent,
  MenubarItem,
  MenubarSeparator,
  MenubarShortcut,
  MenubarSub,
  MenubarSubTrigger,
  MenubarSubContent,
  MenubarCheckboxItem,
  MenubarRadioGroup,
  MenubarRadioItem,
  MenubarLabel,
  MenubarGroup,
} from '@e412/rnui-react'
```

## Usage

```tsx
<Menubar>
  <MenubarMenu>
    <MenubarTrigger>File</MenubarTrigger>
    <MenubarContent>
      <MenubarItem>New Tab <MenubarShortcut>Ctrl+T</MenubarShortcut></MenubarItem>
      <MenubarItem>New Window <MenubarShortcut>Ctrl+N</MenubarShortcut></MenubarItem>
      <MenubarSeparator />
      <MenubarItem>Exit</MenubarItem>
    </MenubarContent>
  </MenubarMenu>
</Menubar>
```

    <MenubarContent>

## Props

### Menubar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |

Accepts all props from Base UI `Menubar`.

### MenubarTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |

### MenubarContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | Content alignment |
| `alignOffset` | `number` | `-4` | Offset from the aligned edge |
| `sideOffset` | `number` | `8` | Distance from the trigger |
| `className` | `string` | — | Additional CSS classes |

### MenubarItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `inset` | `boolean` | — | Whether to add left padding for alignment |
| `variant` | `"default" \| "destructive"` | `"default"` | Visual variant |
| `className` | `string` | — | Additional CSS classes |

### MenubarCheckboxItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Whether the item is checked |
| `inset` | `boolean` | — | Whether to add left padding |
| `className` | `string` | — | Additional CSS classes |

### MenubarShortcut

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |

### MenubarSeparator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
