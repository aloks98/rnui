---
title: Kbd
description: Displays a keyboard key or shortcut in a styled inline element.
---

import { Kbd, KbdGroup } from '@e412/rnui-react';

The `Kbd` component renders a styled keyboard key indicator. Use `KbdGroup` to combine multiple keys into a shortcut sequence.

## Import

```tsx
import { Kbd, KbdGroup } from '@e412/rnui-react';
```

## Usage

<Preview>
  <Kbd>K</Kbd>
</Preview>

```tsx
<Kbd>K</Kbd>
```

## Key Group

Use `KbdGroup` to display multi-key shortcuts.

<Preview>
  <KbdGroup>
    <Kbd>Ctrl</Kbd>
    <Kbd>K</Kbd>
  </KbdGroup>
</Preview>

```tsx
<KbdGroup>
    <Kbd>Ctrl</Kbd>
    <Kbd>K</Kbd>
  </KbdGroup>
```
