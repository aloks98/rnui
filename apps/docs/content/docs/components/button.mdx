---
title: Button
description: A versatile button component with multiple visual variants and sizes, built on Base UI.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { Button, buttonVariants } from '@e412/rnui-react'
```

## Usage

```tsx
<Button>Click me</Button>
```

## Examples

### Variants

```tsx
<Button variant="default">Default</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```

### Sizes

```tsx
<Button size="xs">Extra Small</Button>
<Button size="sm">Small</Button>
<Button size="default">Default</Button>
<Button size="lg">Large</Button>
```

### Icon Buttons

```tsx
import { PlusIcon, SettingsIcon } from 'lucide-react'

<Button size="icon-xs"><PlusIcon /></Button>
<Button size="icon-sm"><PlusIcon /></Button>
<Button size="icon"><SettingsIcon /></Button>
<Button size="icon-lg"><SettingsIcon /></Button>
```

### With Icons

```tsx
import { MailIcon, ArrowRightIcon } from 'lucide-react'

<Button>
  <MailIcon data-icon="inline-start" />
  Send Email
</Button>

<Button variant="outline">
  Next
  <ArrowRightIcon data-icon="inline-end" />
</Button>
```

### Disabled

```tsx
<Button disabled>Disabled</Button>
```

### As Link

```tsx
<Button variant="link" render={<a href="/docs" />}>
  Documentation
</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"` | `"default"` | Visual style variant |
| `size` | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` | Button size |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | — | Additional CSS classes |
| `render` | `React.ReactElement` | — | Custom render element (e.g. `<a>`) via Base UI |
