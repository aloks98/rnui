---
title: Command
description: A command palette with search, keyboard navigation, and grouped actions.
---

import { Command, CommandEmpty, CommandGroup, CommandInput, CommandItem, CommandList, CommandSeparator, CommandShortcut } from '@e412/rnui-react';
import { CalendarIcon, SmileIcon, CalculatorIcon, UserIcon, CreditCardIcon, SettingsIcon, SearchIcon, FileTextIcon, MailIcon, BellIcon } from 'lucide-react';

The `Command` component renders a command palette (built on cmdk) with a search input, grouped items, keyboard shortcuts, and keyboard navigation. It is commonly used for application-wide search or quick-action menus.

## Import

```tsx
import {
  Command,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
  CommandSeparator,
  CommandShortcut,
} from '@e412/rnui-react';
```
