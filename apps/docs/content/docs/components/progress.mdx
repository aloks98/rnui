---
title: Progress
description: A progress bar component with optional label and value display, built on Base UI primitives.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  Progress,
  ProgressLabel,
  ProgressValue,
  ProgressTrack,
  ProgressIndicator,
} from '@e412/rnui-react'
```

## Usage

```tsx
<Progress value={60} />
```

<Preview>
  <Progress value={60} className="w-[60%]" />
</Preview>

## Examples

### With Label and Value

```tsx
<Progress value={45}>
  <ProgressLabel>Uploading...</ProgressLabel>
  <ProgressValue />
</Progress>
```

### Full Width

```tsx
<Progress value={80} className="w-full">
  <ProgressLabel>Storage</ProgressLabel>
  <ProgressValue />
</Progress>
```

### Indeterminate (No Value)

```tsx
<Progress value={null}>
  <ProgressLabel>Loading...</ProgressLabel>
</Progress>
```

## Props

### Progress

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number \| null` | -- | Current progress value (0-100). Pass `null` for indeterminate. |
| `children` | `ReactNode` | -- | Optional label and value elements rendered above the track. |
| `className` | `string` | -- | Additional CSS classes. |

### ProgressLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |
| `children` | `ReactNode` | -- | Label text. |

### ProgressValue

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |

### ProgressTrack

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |

### ProgressIndicator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |
