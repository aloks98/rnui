---
title: ToggleGroup
description: A group of toggle buttons where one or multiple can be selected.
---

import { ToggleGroup, ToggleGroupItem } from '@e412/rnui-react';
import { AlignLeft, AlignCenter, AlignRight, Bold, Italic, Underline } from 'lucide-react';

The `ToggleGroup` component groups toggle buttons together. It supports `single` selection (radio-like) and `multiple` selection (checkbox-like) modes, along with horizontal and vertical orientations.

## Import

```tsx
import { ToggleGroup, ToggleGroupItem } from '@e412/rnui-react';
```

## Usage

<Preview>
  <ToggleGroup type="multiple">
    <ToggleGroupItem value="bold" aria-label="Toggle bold">
      <Bold className="h-4 w-4" />
    </ToggleGroupItem>
    <ToggleGroupItem value="italic" aria-label="Toggle italic">
      <Italic className="h-4 w-4" />
    </ToggleGroupItem>
    <ToggleGroupItem value="underline" aria-label="Toggle underline">
      <Underline className="h-4 w-4" />
    </ToggleGroupItem>
  </ToggleGroup>
</Preview>
