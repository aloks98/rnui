---
title: Slider
description: A draggable slider input for selecting numeric values within a range.
---

import { Slider } from '@e412/rnui-react';

The `Slider` component lets users select a value or range of values along a track by dragging a thumb. It supports single and dual-thumb modes, custom steps, min/max boundaries, and disabled state.

## Import

```tsx
import { Slider } from '@e412/rnui-react';
```

## Usage

<Preview code={`<Slider defaultValue={[50]} max={100} step={1} className="w-[60%]" />`}>
  <Slider defaultValue={[50]} max={100} step={1} className="w-[60%]" />
</Preview>

## Variants

### Range

Use an array with two values to create a range slider with two thumbs.

<Preview code={`<Slider defaultValue={[25, 75]} max={100} step={1} className="w-[60%]" />`}>
  <Slider defaultValue={[25, 75]} max={100} step={1} className="w-[60%]" />
</Preview>

### With Steps

Set `step` to control the increment between selectable values.

<Preview code={`<Slider defaultValue={[20]} max={100} step={10} className="w-[60%]" />`}>
  <Slider defaultValue={[20]} max={100} step={10} className="w-[60%]" />
</Preview>

### Custom Min and Max

<Preview code={`<Slider defaultValue={[500]} min={100} max={1000} step={50} className="w-[60%]" />`}>
  <Slider defaultValue={[500]} min={100} max={1000} step={50} className="w-[60%]" />
</Preview>

### Disabled

<Preview code={`<Slider defaultValue={[50]} max={100} step={1} disabled className="w-[60%]" />`}>
  <Slider defaultValue={[50]} max={100} step={1} disabled className="w-[60%]" />
</Preview>
