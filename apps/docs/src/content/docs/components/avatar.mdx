---
title: Avatar
description: User avatar component with image support, text fallback, status badge, and grouping. Built on Base UI Avatar primitive.
---

## Installation

```bash
npx @e412/rnui-cli add avatar
```

## Import

```tsx
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarBadge,
  AvatarGroup,
  AvatarGroupCount,
} from '@e412/rnui-react'
```

## Usage

```tsx
<Avatar>
  <AvatarImage src="/avatars/user.jpg" alt="User" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```

## Examples

### With Image

```tsx
<Avatar>
  <AvatarImage src="/avatars/01.png" alt="John Doe" />
  <AvatarFallback>JD</AvatarFallback>
</Avatar>
```

### Fallback Only

When no image is provided or the image fails to load, the fallback is displayed.

```tsx
<Avatar>
  <AvatarFallback>AB</AvatarFallback>
</Avatar>
```

### Sizes

```tsx
<Avatar size="sm">
  <AvatarImage src="/avatars/01.png" alt="Small" />
  <AvatarFallback>SM</AvatarFallback>
</Avatar>

<Avatar size="default">
  <AvatarImage src="/avatars/01.png" alt="Default" />
  <AvatarFallback>MD</AvatarFallback>
</Avatar>

<Avatar size="lg">
  <AvatarImage src="/avatars/01.png" alt="Large" />
  <AvatarFallback>LG</AvatarFallback>
</Avatar>
```

### With Badge

```tsx
<Avatar>
  <AvatarImage src="/avatars/01.png" alt="User" />
  <AvatarFallback>JD</AvatarFallback>
  <AvatarBadge />
</Avatar>
```

### Avatar Group

```tsx
<AvatarGroup>
  <Avatar>
    <AvatarImage src="/avatars/01.png" alt="User 1" />
    <AvatarFallback>U1</AvatarFallback>
  </Avatar>
  <Avatar>
    <AvatarImage src="/avatars/02.png" alt="User 2" />
    <AvatarFallback>U2</AvatarFallback>
  </Avatar>
  <Avatar>
    <AvatarImage src="/avatars/03.png" alt="User 3" />
    <AvatarFallback>U3</AvatarFallback>
  </Avatar>
  <AvatarGroupCount>+5</AvatarGroupCount>
</AvatarGroup>
```

### Group with Sizes

```tsx
<AvatarGroup>
  <Avatar size="lg">
    <AvatarFallback>A</AvatarFallback>
  </Avatar>
  <Avatar size="lg">
    <AvatarFallback>B</AvatarFallback>
  </Avatar>
  <AvatarGroupCount>+3</AvatarGroupCount>
</AvatarGroup>
```

## Props

### Avatar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Avatar size |
| `className` | `string` | — | Additional CSS classes |

### AvatarImage

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | — | Image source URL |
| `alt` | `string` | — | Alternative text for the image |
| `className` | `string` | — | Additional CSS classes |

### AvatarFallback

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Fallback content (initials or icon) |
| `className` | `string` | — | Additional CSS classes |

### AvatarBadge

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Optional badge icon |
| `className` | `string` | — | Additional CSS classes |

### AvatarGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Avatar elements |
| `className` | `string` | — | Additional CSS classes |

### AvatarGroupCount

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Count label (e.g. `+5`) |
| `className` | `string` | — | Additional CSS classes |
