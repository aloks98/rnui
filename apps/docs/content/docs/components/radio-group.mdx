---
title: RadioGroup
description: A group of radio buttons where only one option can be selected.
---

import { RadioGroup, RadioGroupItem, Label } from '@e412/rnui-react';

The `RadioGroup` component provides a set of radio buttons for selecting one option from a list. Each item is paired with a `Label` for accessibility.

## Import

```tsx
import { RadioGroup, RadioGroupItem } from '@e412/rnui-react';
```

## Usage

<Preview code={`<RadioGroup defaultValue="comfortable">
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="default" id="r1" />
    <Label htmlFor="r1">Default</Label>
  </div>
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="comfortable" id="r2" />
    <Label htmlFor="r2">Comfortable</Label>
  </div>
  <div className="flex items-center space-x-2">

  </div>
</RadioGroup>`}>
