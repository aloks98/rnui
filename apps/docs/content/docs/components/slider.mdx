---
title: Slider
description: A range slider component for selecting numeric values within a range.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { Slider } from '@e412/rnui-react'
```

## Usage

```tsx
<Slider defaultValue={[50]} max={100} step={1} />
```

<Preview>
  <Slider defaultValue={[50]} max={100} className="w-[60%]" />
</Preview>

## Examples

### Range Slider

```tsx
<Slider defaultValue={[25, 75]} max={100} step={1} />
```

### With Custom Step

```tsx
<Slider defaultValue={[50]} max={100} step={10} />
```

### Vertical Orientation

```tsx
<Slider defaultValue={[50]} max={100} orientation="vertical" />
```

### Disabled

```tsx
<Slider defaultValue={[50]} max={100} disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `number[]` | - | The default slider value(s) |
| `value` | `number[]` | - | Controlled slider value(s) |
| `onValueChange` | `(value: number[]) => void` | - | Callback when the value changes |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Slider orientation |
| `disabled` | `boolean` | `false` | Disable the slider |
| `className` | `string` | - | Additional CSS classes |
