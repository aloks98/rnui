---
title: Sortable
description: A drag-and-drop sortable list built on @dnd-kit for reordering items.
---

import { useState } from 'react';
import { Sortable, SortableItem, SortableItemHandle } from '@e412/rnui-react';
import { GripVertical } from 'lucide-react';

Sortable provides drag-and-drop reordering for lists of items. It uses `@dnd-kit` under the hood and exposes a controlled `value` / `onValueChange` API. Compose it with `SortableItem` for draggable rows and `SortableItemHandle` for dedicated drag handles.

## Import

```tsx
import { Sortable, SortableItem, SortableItemHandle } from '@e412/rnui-react';
```

## Usage

<Preview>
  {() => {
    const [items, setItems] = useState([
      { id: '1', label: 'Item 1' },
      { id: '2', label: 'Item 2' },
      { id: '3', label: 'Item 3' },
      { id: '4', label: 'Item 4' },
      { id: '5', label: 'Item 5' },
    ]);
    return (
      <Sortable
        value={items}
        onValueChange={setItems}
        getItemValue={(item) => item.id}
        className="flex flex-col gap-2 w-72"
      >
        {items.map((item) => (
          <SortableItem key={item.id} value={item.id}>
            <SortableItemHandle className="flex items-center gap-2 rounded-lg border bg-card p-3 cursor-grab">
              <GripVertical className="h-4 w-4 text-muted-foreground" />
              <span className="text-sm text-foreground">{item.label}</span>
            </SortableItemHandle>
          </SortableItem>
        ))}
      </Sortable>
    );
  }}
</Preview>

## With Separate Handles

Use `SortableItemHandle` as a child of `SortableItem` to restrict the drag interaction to just the handle icon, allowing the rest of the row to remain interactive.

      </div>
    );
