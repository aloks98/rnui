---
title: Button
description: A clickable element that triggers an action or event.
---

import { Button } from '@e412/rnui-react';
import { Plus, Settings, Loader2, ExternalLink, Mail } from 'lucide-react';

The `Button` component is used to trigger actions. It supports multiple variants, sizes, and can render as different HTML elements or links.

## Import

```tsx
import { Button } from '@e412/rnui-react';
```

## Usage

<Preview>
  <Button>Button</Button>
</Preview>

```tsx
<Button>Button</Button>
```
