---
title: Sheet
description: A side panel overlay that slides in from any edge of the screen.
---

import { Sheet, SheetClose, SheetContent, SheetDescription, SheetFooter, SheetHeader, SheetTitle, SheetTrigger, Button, Input, Label } from '@e412/rnui-react';
import { Settings } from 'lucide-react';

The `Sheet` component is a panel that slides in from the edge of the screen, overlaying the page content. It is ideal for navigation, filters, settings, and supplementary forms.

## Import

```tsx
import {
  Sheet,
  SheetClose,
  SheetContent,
  SheetDescription,
  SheetFooter,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <Sheet>
    <SheetTrigger render={<Button variant="outline" />}>
      Open Sheet
    </SheetTrigger>
    <SheetContent>
      <SheetHeader>
        <SheetTitle>Edit Profile</SheetTitle>
        <SheetDescription>
          Make changes to your profile here. Click save when you are done.
        </SheetDescription>
      </SheetHeader>
      <div className="p-4">
        <p className="text-sm text-muted-foreground">Sheet body content.</p>
      </div>
      <SheetFooter>
        <SheetClose render={<Button type="submit" />}>
          Save changes
        </SheetClose>
      </SheetFooter>
    </SheetContent>
  </Sheet>
</Preview>

## Sides
