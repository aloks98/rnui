---
title: Card
description: A container component for grouping related content with header, body, and footer sections.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
  CardAction,
} from '@e412/rnui-react'
```

## Usage

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card description goes here.</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card content goes here.</p>
  </CardContent>
  <CardFooter>
    <Button>Action</Button>
  </CardFooter>
</Card>
```

## Examples

### Simple Card

```tsx
<Card className="w-[350px]">
  <CardContent>
    <p>A simple card with just content.</p>
  </CardContent>
</Card>
```

### Small Size

```tsx
<Card size="sm" className="w-[350px]">
  <CardHeader>
    <CardTitle>Compact Card</CardTitle>
    <CardDescription>Uses smaller padding and gaps.</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Content with tighter spacing.</p>
  </CardContent>
</Card>
```

### With Card Action

```tsx
<Card className="w-[350px]">
  <CardHeader>
    <CardTitle>Notifications</CardTitle>
    <CardDescription>Manage your notification preferences.</CardDescription>
    <CardAction>
      <Button variant="outline" size="sm">Settings</Button>
    </CardAction>
  </CardHeader>
  <CardContent>
    <p>You have 3 unread notifications.</p>
  </CardContent>
  <CardFooter>
    <Button>Mark all as read</Button>
  </CardFooter>
</Card>
```

### With Image

```tsx
<Card className="w-[350px]">
  <img src="/landscape.jpg" alt="Landscape" className="w-full" />
  <CardHeader>
    <CardTitle>Beautiful Scenery</CardTitle>
    <CardDescription>A card with a top image.</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Images placed as the first child are automatically rounded at the top.</p>
  </CardContent>
</Card>
```

## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Controls padding and gap sizing |
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Card contents |

### CardHeader

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Header contents (typically CardTitle, CardDescription, CardAction) |

### CardTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Title text |

### CardDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Description text |

### CardAction

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Action content, positioned at the top-right of CardHeader |

### CardContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Main body content |

### CardFooter

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | Footer content, rendered with a top border and muted background |
