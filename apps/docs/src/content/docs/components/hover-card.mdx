---
title: Hover Card
description: A card that appears when hovering over a trigger element, built on Base UI PreviewCard.
---

## Installation

```bash
npx @e412/rnui-cli add hover-card
```

## Import

```tsx
import {
  HoverCard,
  HoverCardTrigger,
  HoverCardContent,
} from '@e412/rnui-react'
```

## Usage

```tsx
<HoverCard>
  <HoverCardTrigger render={<a href="/user/johndoe" />}>
    @johndoe
  </HoverCardTrigger>
  <HoverCardContent>
    <div className="flex gap-3">
      <img
        src="/avatars/johndoe.png"
        alt="John Doe"
        className="size-10 rounded-full"
      />
      <div className="flex flex-col gap-1">
        <p className="text-sm font-medium">John Doe</p>
        <p className="text-xs text-muted-foreground">
          Full-stack developer. Open source enthusiast.
        </p>
      </div>
    </div>
  </HoverCardContent>
</HoverCard>
```

## Examples

### Custom Positioning

```tsx
<HoverCard>
  <HoverCardTrigger render={<a href="#" />}>
    Hover for details
  </HoverCardTrigger>
  <HoverCardContent side="right" align="start" sideOffset={8}>
    <p className="text-sm">
      This card appears to the right of the trigger.
    </p>
  </HoverCardContent>
</HoverCard>
```

### Rich Content

```tsx
<HoverCard>
  <HoverCardTrigger render={<a href="/project/rnui" className="underline" />}>
    rnui
  </HoverCardTrigger>
  <HoverCardContent>
    <div className="flex flex-col gap-2">
      <p className="text-sm font-medium">rnui</p>
      <p className="text-xs text-muted-foreground">
        A modern React component library with beautiful defaults.
      </p>
      <div className="flex items-center gap-2 text-xs text-muted-foreground">
        <span>TypeScript</span>
        <span>342 stars</span>
        <span>Updated 2 days ago</span>
      </div>
    </div>
  </HoverCardContent>
</HoverCard>
```

## Props

### HoverCard

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | — | Callback when open state changes |
| `defaultOpen` | `boolean` | `false` | Initial open state (uncontrolled) |
| `delay` | `number` | — | Delay before showing (ms) |

### HoverCardTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `render` | `React.ReactElement` | — | Custom render element for composition (Base UI pattern) |
| `className` | `string` | — | Additional CSS classes |

### HoverCardContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Side relative to trigger |
| `sideOffset` | `number` | `4` | Offset from trigger |
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment relative to trigger |
| `alignOffset` | `number` | `4` | Offset from alignment |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Card contents |
