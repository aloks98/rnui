---
title: HoverCard
description: A card that appears on hover to show additional context about an element.
---

import { HoverCard, HoverCardContent, HoverCardTrigger, Button, Avatar, AvatarImage, AvatarFallback } from '@e412/rnui-react';
import { CalendarDays } from 'lucide-react';

The `HoverCard` component displays a floating card with rich content when the user hovers over a trigger element. It is useful for showing previews, profile cards, or supplementary details without navigating away.

## Import

```tsx
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <HoverCard>
    <HoverCardTrigger render={<Button variant="link" />}>
      @nextjs
    </HoverCardTrigger>
    <HoverCardContent className="w-80">
      <div className="space-y-1">
        <h4 className="text-sm font-semibold">@nextjs</h4>
        <p className="text-sm text-muted-foreground">
          The React Framework for the Web. Created and maintained by @vercel.
        </p>
        <p className="text-xs text-muted-foreground">Joined December 2021</p>
      </div>
    </HoverCardContent>
  </HoverCard>
</Preview>
