---
title: Resizable
description: Resizable panel groups for building adjustable layouts.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from '@e412/rnui-react'
```

## Usage

```tsx
<ResizablePanelGroup direction="horizontal">
  <ResizablePanel>Panel One</ResizablePanel>
  <ResizableHandle />
  <ResizablePanel>Panel Two</ResizablePanel>
</ResizablePanelGroup>
```

## Examples

### Horizontal Layout

```tsx
<ResizablePanelGroup direction="horizontal" className="min-h-[200px] rounded-lg border">
  <ResizablePanel defaultSize={50}>
    <div className="flex h-full items-center justify-center p-6">
      <span>Left Panel</span>
    </div>
  </ResizablePanel>
  <ResizableHandle />
  <ResizablePanel defaultSize={50}>
    <div className="flex h-full items-center justify-center p-6">
      <span>Right Panel</span>
    </div>
  </ResizablePanel>
</ResizablePanelGroup>
```

## Props

### ResizablePanelGroup

Extends `react-resizable-panels` `GroupProps`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `"horizontal" \| "vertical"` | -- | The direction panels are laid out |
| `className` | `string` | -- | Additional CSS classes |
| `children` | `ReactNode` | -- | ResizablePanel and ResizableHandle elements |

### ResizablePanel

Extends `react-resizable-panels` `PanelProps`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultSize` | `number` | -- | Initial size as a percentage (0-100) |
| `minSize` | `number` | -- | Minimum size as a percentage |
| `maxSize` | `number` | -- | Maximum size as a percentage |
| `collapsible` | `boolean` | `false` | Whether the panel can be collapsed |

### ResizableHandle

Extends `react-resizable-panels` `SeparatorProps`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `withHandle` | `boolean` | `false` | Show a visible drag handle indicator |
| `className` | `string` | -- | Additional CSS classes |
