title: Tree
description: A tree view component powered by headless-tree for rendering hierarchical data with expand/collapse, selection, drag-and-drop, and search support.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes @headless-tree/core @headless-tree/react
```

## Import

```tsx
import { Tree, TreeItem, TreeItemLabel, TreeDragLine } from '@e412/rnui-react'
```

## Usage

```tsx
import { useTree } from '@headless-tree/react'

const items = {
  root: { name: "Root", children: ["src", "public"] },
  src: { name: "src", children: ["app", "lib"] },
  public: { name: "public", children: ["favicon"] },
  app: { name: "app.tsx" },
  lib: { name: "lib.ts" },
  favicon: { name: "favicon.ico" },
}

function MyTree() {
  const tree = useTree({
    rootItemId: "root",
    getItemName: (item) => item.name,
    isItemFolder: (item) => !!item.children,
    dataLoader: {
      getItem: (id) => items[id],
      getChildren: (id) => items[id].children ?? [],
    },
  })

  return (
    <Tree tree={tree}>
      {tree.getItems().map((item) => (
        <TreeItem key={item.getId()} item={item}>
          <TreeItemLabel />
        </TreeItem>
      ))}
    </Tree>
  )
}
```

## Examples

### Chevron Toggle Icons

```tsx
<Tree tree={tree} toggleIconType="chevron">
  {tree.getItems().map((item) => (
    <TreeItem key={item.getId()} item={item}>
      <TreeItemLabel />
    </TreeItem>
  ))}
</Tree>
```

## Props

### Tree

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `tree` | `TreeInstance` | -- | Tree instance from `useTree()` (headless-tree). |
| `indent` | `number` | `20` | Pixel indentation per nesting level. |
| `toggleIconType` | `"chevron" \| "plus-minus"` | `"chevron"` | Style of the expand/collapse icon for folder items. |
| `className` | `string` | -- | Additional CSS classes. |

### TreeItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `item` | `ItemInstance<T>` | -- | **Required.** The headless-tree item instance. |
| `className` | `string` | -- | Additional CSS classes. |
| `render` | `RenderProp` | -- | Custom render function via Base UI `useRender`. |
| `children` | `ReactNode` | -- | Item contents (typically a `TreeItemLabel`). |

### TreeItemLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `item` | `ItemInstance<T>` | -- | Item instance. Falls back to context if omitted. |
| `className` | `string` | -- | Additional CSS classes. |
| `children` | `ReactNode` | -- | Custom label content. Defaults to `item.getItemName()`. |

### TreeDragLine

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |
