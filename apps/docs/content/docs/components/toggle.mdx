---
title: Toggle
description: A two-state toggle button that can be pressed or unpressed.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { Toggle } from '@e412/rnui-react'
```

## Usage

```tsx
<Toggle aria-label="Toggle bold">
  <BoldIcon className="size-4" />
</Toggle>
```

## Examples

### Outline Variant

```tsx
<Toggle variant="outline" aria-label="Toggle italic">
  <ItalicIcon className="size-4" />
</Toggle>
```

### With Text

```tsx
<Toggle aria-label="Toggle italic">
  <ItalicIcon className="size-4" />
  Italic
</Toggle>
```

### Sizes

```tsx
<Toggle size="sm" aria-label="Small">
  <BoldIcon className="size-4" />
</Toggle>
<Toggle size="default" aria-label="Default">
  <BoldIcon className="size-4" />
</Toggle>
<Toggle size="lg" aria-label="Large">
  <BoldIcon className="size-4" />
</Toggle>
```

### Disabled

```tsx
<Toggle disabled aria-label="Toggle disabled">
  <BoldIcon className="size-4" />
</Toggle>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "outline"` | `"default"` | Visual style variant |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Toggle size |
| `pressed` | `boolean` | - | Controlled pressed state |
| `defaultPressed` | `boolean` | `false` | Default pressed state |
| `onPressedChange` | `(pressed: boolean) => void` | - | Callback when pressed state changes |
| `disabled` | `boolean` | `false` | Disable the toggle |
| `className` | `string` | - | Additional CSS classes |
