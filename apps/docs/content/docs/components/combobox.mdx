---
title: Combobox
description: A searchable dropdown for selecting from a list of options.
---

import { Combobox, ComboboxInput, ComboboxContent, ComboboxList, ComboboxItem, ComboboxGroup, ComboboxLabel, ComboboxEmpty, ComboboxSeparator } from '@e412/rnui-react';

The `Combobox` component combines a text input with a filterable dropdown list. Users can type to search and filter options, then select a value. It supports groups, separators, empty states, and a clear button.

## Import

```tsx
import {
  Combobox,
  ComboboxInput,
  ComboboxContent,
  ComboboxList,
  ComboboxItem,
  ComboboxGroup,
  ComboboxLabel,
  ComboboxEmpty,
  ComboboxSeparator,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <Combobox items={['Apple', 'Banana', 'Blueberry', 'Grapes', 'Pineapple']}>
    <ComboboxInput placeholder="Search fruits..." />
    <ComboboxContent>
      <ComboboxEmpty>No results found.</ComboboxEmpty>
      <ComboboxList>
        {(item) => (
          <ComboboxItem key={item} value={item}>
            {item}
          </ComboboxItem>
        )}
      </ComboboxList>
    </ComboboxContent>
  </Combobox>
</Preview>

