---
title: Sidebar
description: A collapsible side navigation panel with support for grouped menus, headers, and footers.
---

import {
  Sidebar,
  SidebarContent,
  SidebarFooter,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarHeader,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem,
  SidebarProvider,
  SidebarTrigger,
  SidebarSeparator,
  SidebarMenuSub,
  SidebarMenuSubButton,
  SidebarMenuSubItem,
  SidebarInset,
} from '@e412/rnui-react';
import { Home, Settings, User, FileText, BarChart3, Inbox, Search, ChevronDown, LogOut, Bell, HelpCircle } from 'lucide-react';

The `Sidebar` component provides a collapsible side navigation panel. Wrap your layout with `SidebarProvider` and use `SidebarTrigger` to toggle visibility. The sidebar supports grouped menus, headers, footers, and collapses to an icon-only rail on desktop or a sheet on mobile.

## Import

```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarFooter,
  SidebarGroup,
  SidebarGroupContent,
  SidebarGroupLabel,
  SidebarHeader,
  SidebarMenu,
  SidebarMenuButton,
  SidebarMenuItem,
  SidebarProvider,
  SidebarTrigger,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <SidebarProvider>
    <div className="flex min-h-[400px] w-full">
      <Sidebar>

          <div className="flex items-center gap-2 px-2 py-1.5">

          </div>

        <SidebarContent>
          <SidebarGroup>
            <SidebarGroupLabel>Navigation</SidebarGroupLabel>
            <SidebarGroupContent>
              <SidebarMenu>
                <SidebarMenuItem>
                  <SidebarMenuButton>
                    <Home className="size-4" />
                    Home

                </SidebarMenuItem>

                    <Inbox className="size-4" />

                  <SidebarMenuButton>
                    <Settings className="size-4" />
