---
title: Kbd
description: Displays a keyboard key or shortcut in a styled inline element.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { Kbd, KbdGroup } from '@e412/rnui-react'
```

## Usage

```tsx
<Kbd>Ctrl</Kbd>
```

## Examples

### Single Key

```tsx
<Kbd>Esc</Kbd>
```

### Key Combination with KbdGroup

```tsx
<KbdGroup>
  <Kbd>Ctrl</Kbd>
  <Kbd>S</Kbd>
</KbdGroup>
```

### Multiple Shortcuts

```tsx
<KbdGroup>
  <Kbd>Cmd</Kbd>
  <Kbd>Shift</Kbd>
  <Kbd>P</Kbd>
</KbdGroup>
```

### Inside Text

```tsx
<p className="text-sm text-muted-foreground">
  Press <Kbd>Ctrl</Kbd> + <Kbd>K</Kbd> to open the command palette.
</p>
```

## Props

### Kbd

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |
| `children` | `ReactNode` | -- | Key label text or icon. |

### KbdGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. |
| `children` | `ReactNode` | -- | Multiple `Kbd` elements to display together. |
