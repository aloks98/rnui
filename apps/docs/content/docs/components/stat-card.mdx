---
title: StatCard
description: A card displaying a key metric with optional icon and trend indicator.
---

import { StatCard } from '@e412/rnui-react';
import { DollarSign, Users, ShoppingCart, TrendingUp } from 'lucide-react';

The `StatCard` component presents a single metric with a title, value, description, optional icon, and trend indicator showing positive or negative change.

## Import

```tsx
import { StatCard } from '@e412/rnui-react';
```

## Usage

<Preview code={`<StatCard
    title="Total Revenue"
    value="$45,231.89"
    description="Monthly revenue"
  />`}>
  <StatCard
    title="Total Revenue"
    value="$45,231.89"
    description="Monthly revenue"
  />
</Preview>

## With Trend

<Preview code={`<StatCard
    title="Total Revenue"
    value="$45,231.89"
    description="+20.1% from last month"
    trend={{ value: 20.1, isPositive: true }}
  />`}>
  <StatCard
    title="Total Revenue"
    value="$45,231.89"
    description="+20.1% from last month"
    trend={{ value: 20.1, isPositive: true }}
  />
</Preview>

## Negative Trend

<Preview code={`<StatCard
