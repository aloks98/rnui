---
title: Autocomplete
description: A searchable dropdown input with filtering, built on Base UI's Autocomplete primitive.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  Autocomplete,
  AutocompleteInput,
  AutocompleteContent,
  AutocompleteList,
  AutocompleteItem,
  AutocompleteEmpty,
  AutocompleteGroup,
  AutocompleteGroupLabel,
  AutocompleteSeparator,
} from '@e412/rnui-react'
```

## Usage

```tsx
<Autocomplete>
  <AutocompleteInput placeholder="Search fruits..." showTrigger />
  <AutocompleteContent>
    <AutocompleteList>
      <AutocompleteItem value="apple">Apple</AutocompleteItem>
      <AutocompleteItem value="banana">Banana</AutocompleteItem>
      <AutocompleteItem value="cherry">Cherry</AutocompleteItem>
    </AutocompleteList>
    <AutocompleteEmpty>No results found.</AutocompleteEmpty>
  </AutocompleteContent>
</Autocomplete>
```

## Props

### Autocomplete

The root component. Accepts all `Base UI Autocomplete.Root` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled selected value |
| `onValueChange` | `(value: string) => void` | — | Callback when value changes |
| `defaultValue` | `string` | — | Initial value (uncontrolled) |

### AutocompleteInput

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Input height |
| `showClear` | `boolean` | `false` | Show clear button |
| `showTrigger` | `boolean` | `false` | Show dropdown trigger icon |
| `placeholder` | `string` | — | Placeholder text |
| `className` | `string` | — | Additional CSS classes |

### AutocompleteContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"start"` | Popup alignment |
| `side` | `"top" \| "bottom"` | `"bottom"` | Popup side |
| `sideOffset` | `number` | `4` | Distance from anchor |
| `alignOffset` | `number` | `0` | Alignment offset |
| `showBackdrop` | `boolean` | `false` | Show backdrop overlay |
| `className` | `string` | — | Additional CSS classes |

### AutocompleteItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Item value |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Item content |

### AutocompleteEmpty

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Empty state content |
