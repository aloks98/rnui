## Packages

| Package | Description |
|---------|-------------|
| `@e412/rnui-react` | React components, hooks, and utilities |
| `@e412/rnui-themes` | CSS theme with design tokens and Tailwind v4 integration |

## Installation

Install the React components and themes packages:

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

Or with pnpm:

```bash
pnpm add @e412/rnui-react @e412/rnui-themes
```

### Charts (Optional)

If you plan to use chart components, install ECharts as a peer dependency:

```bash
npm install echarts
```

## CSS Setup

Import the theme CSS in your app's global stylesheet:

```css
@import '@e412/rnui-themes';
@import 'tw-animate-css';
```

## Tailwind v4 Integration

The theme CSS includes a `@theme inline` block that automatically registers all design tokens with Tailwind v4. Just importing `@e412/rnui-themes` gives you utilities like `bg-primary`, `text-muted-foreground`, etc.

Add the library to your Tailwind content sources:

```css
@source "../node_modules/@e412/rnui-react/dist";
```

## Dark Mode

Toggle dark mode by adding the `dark` class to your `<html>` element:

```html
<html class="dark">
```

The theme automatically switches all color tokens.

## Quick Example

Once installed, import and use components:

```tsx
import { Button } from '@e412/rnui-react'

export function MyApp() {
  return <Button>Click me</Button>
}
```

### Importing Multiple Components

```tsx
import {
  Button,
  Card,
  CardHeader,
  CardTitle,
  CardContent,
  Input,
  Label,
} from '@e412/rnui-react'
```

### Using Hooks

```tsx
import { useIsMobile, useFileUpload } from '@e412/rnui-react'
```

### Using Charts

```tsx
import { BarChart, LineChart, PieChart, chartConfig } from '@e412/rnui-react'
```

### Using Utilities

```tsx
import { cn } from '@e412/rnui-react'

cn('px-4 py-2', isActive && 'bg-primary', className)
```

That's it — your components are ready to use.
