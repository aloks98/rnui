---
title: DateSelector
description: A flexible date selection component with period types, range mode, and text input.
---

import { DateSelector } from '@e412/rnui-react';

The `DateSelector` component provides a rich date selection interface with support for single dates, date ranges, month/quarter/year period types, optional text input, and single or dual-month calendar views.

## Import

```tsx
import { DateSelector } from '@e412/rnui-react';
```

## Usage

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector />
  </div>
</Preview>

## Variants

### With Label

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector label="Select Date" />
  </div>
</Preview>

### Range Mode

Set `defaultFilterType="between"` to enable date range selection.

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector defaultFilterType="between" />
  </div>
</Preview>

### Month Only

Restrict selection to months using the `periodTypes` prop.

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector periodTypes={["month"]} />
  </div>
</Preview>

### Quarter and Year

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector periodTypes={["quarter", "year"]} />
  </div>
</Preview>

### Without Text Input

Hide the text input with `showInput={false}`.

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector showInput={false} />
  </div>
</Preview>

### Single Month View

Show only one month at a time with `showTwoMonths={false}`.

<Preview>
  <div className="w-full max-w-xl">
    <DateSelector showTwoMonths={false} />
  </div>
</Preview>
