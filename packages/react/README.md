# @e412/rnui-react

All shadcn/ui components, packaged for reuse across projects.

## Install

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Peer Dependencies

- `react` >= 18
- `react-dom` >= 18
- `tailwindcss` >= 4
- `@e412/rnui-themes`

## Usage

```tsx
import { Button } from '@e412/rnui-react'
```

See the [root README](../../README.md) for full setup instructions.
