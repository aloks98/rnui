---
title: Empty State
description: A placeholder component for empty views, featuring an icon, title, description, and optional action button.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { EmptyState } from '@e412/rnui-react'
```

## Usage

```tsx
import { InboxIcon } from 'lucide-react'
import { Button } from '@e412/rnui-react'

<EmptyState
  icon={<InboxIcon />}
  title="No messages"
  description="You don't have any messages yet."
  action={<Button>Compose</Button>}
/>
```

## Examples

### Basic

```tsx
<EmptyState title="No results found" />
```

### With Icon and Description

```tsx
import { FileIcon } from 'lucide-react'

<EmptyState
  icon={<FileIcon />}
  title="No documents"
  description="Upload a document to get started."
/>
```

### With Action

```tsx
import { PlusIcon } from 'lucide-react'
import { Button } from '@e412/rnui-react'

<EmptyState
  icon={<PlusIcon />}
  title="No projects"
  description="Create your first project to begin."
  action={<Button>Create Project</Button>}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | -- | Icon displayed above the title. SVGs are automatically sized to 48x48. |
| `title` | `string` | -- | **Required.** Heading text for the empty state. |
| `description` | `string` | -- | Supporting text below the title. |
| `action` | `ReactNode` | -- | Action element (e.g. a Button) rendered below the description. |
| `className` | `string` | -- | Additional CSS classes. |
