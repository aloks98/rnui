import { LineChart } from '@e412/rnui-react';

The `LineChart` component renders single or multi-line charts. It supports smooth curves, custom tooltips, and annotations via the `option` prop.

## Import

```tsx
import { LineChart } from '@e412/rnui-react';
```

## Annotations

Use the `option` prop to add mark points and mark lines for min, max, and average annotations.

<Preview>
  <div className="w-full max-w-lg">
    <LineChart
      data={[
        { name: 'Mon', value: 8200 },
        { name: 'Tue', value: 9320 },
        { name: 'Wed', value: 9010 },
        { name: 'Thu', value: 12340 },
        { name: 'Fri', value: 12900 },
        { name: 'Sat', value: 15300 },
        { name: 'Sun', value: 13200 },
      ]}
      smooth
      height={350}
      option={{
        series: [
          {
            type: 'line',
            data: [8200, 9320, 9010, 12340, 12900, 15300, 13200],
            smooth: true,
            symbolSize: 6,
            markPoint: {
              data: [
                { type: 'max', name: 'Max' },
                { type: 'min', name: 'Min' },
              ],
              label: { fontSize: 10 },
            },
            markLine: {
              data: [{ type: 'average', name: 'Average' }],
              label: { formatter: 'Avg: {c}', fontSize: 11 },
              lineStyle: { type: 'dashed' },
            },
          },
        ],
        xAxis: { type: 'category', data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'], axisTick: { show: false } },
        yAxis: { type: 'value', splitLine: { lineStyle: { type: 'dashed', opacity: 0.5 } } },
        grid: { containLabel: true, left: 16, right: 16, top: 40, bottom: 8 },
      }}
    />
  </div>
</Preview>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `{ name: string; value: number }[]` | -- | Data items for single-line charts |
| `categories` | `string[]` | -- | X-axis category labels (for multi-series) |
| `series` | `{ name: string; data: number[] }[]` | -- | Series data for multi-line charts |
| `smooth` | `boolean` | `false` | Use smooth bezier curves |
| `showLegend` | `boolean` | `false` | Show chart legend |
| `height` | `number` | `350` | Chart height in pixels |
| `loading` | `boolean` | `false` | Show loading state |
| `animateOnMount` | `boolean` | `true` | Animate on first render |
| `option` | `object` | -- | Override or extend the ECharts option |
