---
title: Tabs
description: Organise content into switchable panels triggered by a tab list.
---

import { Tabs, TabsContent, TabsList, TabsTrigger, Card, CardContent } from '@e412/rnui-react';
import { Settings, User, Lock } from 'lucide-react';

The `Tabs` component lets users switch between views within a shared context. It supports horizontal and vertical orientations and two visual variants.

## Import

```tsx
import { Tabs, TabsContent, TabsList, TabsTrigger } from '@e412/rnui-react';
```

## Usage

<Preview>
  <Tabs defaultValue="account" className="w-[400px]">
    <TabsList>
      <TabsTrigger value="account">Account</TabsTrigger>
      <TabsTrigger value="password">Password</TabsTrigger>
    </TabsList>
    <TabsContent value="account">
      <p className="text-sm text-muted-foreground">Make changes to your account here.</p>
    </TabsContent>
    <TabsContent value="password">
      <p className="text-sm text-muted-foreground">Change your password here.</p>
    </TabsContent>
  </Tabs>
</Preview>
