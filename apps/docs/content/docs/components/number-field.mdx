---
title: Number Field
description: A number input with increment and decrement buttons, scrub area support, and size variants.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import {
  NumberField,
  NumberFieldGroup,
  NumberFieldInput,
  NumberFieldIncrement,
  NumberFieldDecrement,
  NumberFieldScrubArea,
} from '@e412/rnui-react'
```

## Usage

```tsx
<NumberField defaultValue={0}>
  <NumberFieldGroup>
    <NumberFieldDecrement />
    <NumberFieldInput />
    <NumberFieldIncrement />
  </NumberFieldGroup>
</NumberField>
```

<Preview>

</Preview>

```tsx
  </NumberFieldGroup>
```

### Disabled

```tsx

  <NumberFieldGroup>
    <NumberFieldDecrement />
```

### Controlled

```tsx

<NumberField value={value} onValueChange={setValue}>

    <NumberFieldDecrement />
    <NumberFieldInput />
    <NumberFieldIncrement />
  </NumberFieldGroup>
</NumberField>
```

## Props

### NumberField

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `number` | - | Initial uncontrolled value |
| `value` | `number` | - | Controlled value |
| `onValueChange` | `(value: number) => void` | - | Called when value changes |
| `min` | `number` | - | Minimum allowed value |
| `max` | `number` | - | Maximum allowed value |
| `step` | `number` | `1` | Step increment |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | Size variant |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | - | Additional CSS class |

### NumberFieldGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg"` | Inherited | Override size from parent |
| `className` | `string` | - | Additional CSS class |

### NumberFieldInput

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg"` | Inherited | Override size from parent |
| `className` | `string` | - | Additional CSS class |

### NumberFieldIncrement / NumberFieldDecrement

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg"` | Inherited | Override size from parent |
| `children` | `React.ReactNode` | Plus/Minus icon | Custom button content |
| `className` | `string` | - | Additional CSS class |

### NumberFieldScrubArea

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Label text (required for accessibility) |
| `className` | `string` | - | Additional CSS class |
