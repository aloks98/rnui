## v0.0.1

Initial release of the @e412/rnui component library.

### Highlights

- **60+ components** built on [Base UI](https://base-ui.com/) and [shadcn/ui](https://ui.shadcn.com/) patterns
- **Theme system** with `@e412/rnui-themes` -- CSS custom properties in oklch, light/dark mode, Tailwind v4 integration
- **ECharts integration** -- BarChart, LineChart, AreaChart, PieChart, RadarChart, ScatterChart with automatic theme syncing
- **chartConfig() builder** -- fluent API for composing chart options
- **DataGrid** -- full-featured data table with sorting, filtering, pagination, column visibility, drag-and-drop columns/rows, and virtualization
- **DateSelector** -- date selection with calendar integration
- **Filters** -- configurable filter controls
- **Sortable** -- drag-and-drop sortable lists
- **Form integration** -- react-hook-form compatible Form components
- **Tree-shakeable** -- import only what you use, all dependencies externalized
