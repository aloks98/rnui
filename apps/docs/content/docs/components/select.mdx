---
title: Select
description: A dropdown menu for selecting a value from a list of options.
---

import { Select, SelectContent, SelectGroup, SelectItem, SelectLabel, SelectTrigger, SelectValue, Label } from '@e412/rnui-react';

The `Select` component provides a dropdown for choosing from a list of options. It uses a composed API with trigger, content, groups, and items.

## Import

```tsx
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectItem,
  SelectLabel,
  SelectTrigger,
  SelectValue,
} from '@e412/rnui-react';
```

## Usage

      <SelectItem value="apple">Apple</SelectItem>
      <SelectItem value="banana">Banana</SelectItem>
