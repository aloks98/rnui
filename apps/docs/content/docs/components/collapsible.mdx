---
title: Collapsible
description: A component that expands and collapses content.
---

import { Collapsible, CollapsibleTrigger, CollapsibleContent, Button } from '@e412/rnui-react';

The `Collapsible` component allows you to toggle the visibility of content. It consists of a trigger element and a content section that can be expanded or collapsed.

## Import

```tsx
import {
  Collapsible,
  CollapsibleTrigger,
  CollapsibleContent,
} from '@e412/rnui-react';
```

## Usage

<Preview code={`<Collapsible className="w-[350px] space-y-2">

      Toggle
