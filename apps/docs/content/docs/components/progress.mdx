---
title: Progress
description: A horizontal bar that indicates the completion status of a task.
---

import { Progress, ProgressLabel, ProgressValue } from '@e412/rnui-react';

The `Progress` component displays a horizontal bar that fills to represent completion percentage. Use it for uploads, downloads, multi-step flows, or any measurable process.

## Import

```tsx
import { Progress, ProgressLabel, ProgressValue } from '@e412/rnui-react';
```

## Usage

<Preview code={`<Progress value={60} className="w-[60%]" />`}>
  <Progress value={60} className="w-[60%]" />
</Preview>

    <Progress value={0} />

## With Label and Value

Use `ProgressLabel` and `ProgressValue` to display text alongside the progress bar.

<Preview code={`<Progress value={60} className="w-[60%]">
    <ProgressLabel>Uploading...</ProgressLabel>
    <ProgressValue />
  </Progress>`}>
  <Progress value={60} className="w-[60%]">
    <ProgressLabel>Uploading...</ProgressLabel>
    <ProgressValue />
  </Progress>
</Preview>
