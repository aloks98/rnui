import { EChart, echarts, chartConfig } from '@e412/rnui-react';

The `EChart` component is the base chart wrapper that accepts a raw Apache ECharts `option` object. Use it when you need chart types or configurations not covered by the high-level components, such as gauges, heatmaps, treemaps, or mixed chart types.

## Import

```tsx
import { EChart, echarts } from '@e412/rnui-react';
```

## Usage

Pass any valid ECharts option object directly.

<Preview>
  <div className="w-full max-w-lg">

        tooltip: { trigger: 'axis' },
      }}
      height={300}
    />
  </div>
</Preview>

## Using `chartConfig()` Builder

The fluent builder API provides a convenient way to construct options.

<Preview>
  <div className="w-full max-w-lg">
    <EChart
      option={chartConfig()
        .line({ data: [4200, 3800, 5100, 4600, 5800, 6200], categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'], name: 'Revenue', smooth: true })
        .line({ data: [2800, 3200, 2900, 3100, 2700, 3500], name: 'Expenses', smooth: true })
        .legend()
        .tooltip()
        .build()}
      height={350}
    />
  </div>
</Preview>

```tsx
<div className="w-full max-w-lg">
    <EChart
      option={chartConfig()
        .line({ data: [4200, 3800, 5100, 4600, 5800, 6200], categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'], name: 'Revenue', smooth: true })
        .line({ data: [2800, 3200, 2900, 3100, 2700, 3500], name: 'Expenses', smooth: true })
        .legend()
        .tooltip()
        .build()}
      height={350}
    />
  </div>
```

```tsx
const option = chartConfig()
  .line({ data: [4200, 3800, 5100], categories: ['Jan', 'Feb', 'Mar'], name: 'Revenue', smooth: true })
  .line({ data: [2800, 3200, 2900], name: 'Expenses', smooth: true })
  .legend()
  .tooltip()
  .build();

<EChart option={option} height={350} />
```

## Disabling Presets

Set `preset={false}` to use raw ECharts defaults without shadcn-style presets. Useful for sparklines or highly custom layouts.

          xAxis: { show: false, type: 'category', data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'] },
          yAxis: { show: false, type: 'value' },
          series: [{
            type: 'line',
            data: [820, 932, 901, 1234, 1290, 1530, 1320],

## Extending with Custom Chart Types

Import additional ECharts chart types via the `echarts` instance.

```tsx
import { EChart, echarts } from '@e412/rnui-react';
import { GaugeChart } from 'echarts/charts';

echarts.use([GaugeChart]);

<EChart
  option={{
    series: [{
      type: 'gauge',
      data: [{ value: 72, name: 'CPU Usage' }],
    }],
  }}
  height={350}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `option` | `EChartsCoreOption` | required | ECharts option object |
| `height` | `number \| string` | `350` | Chart height |
| `width` | `number \| string` | `"100%"` | Chart width |
| `theme` | `string` | auto | Theme name (auto-detects light/dark) |
| `preset` | `boolean` | `true` | Apply shadcn-style minimal defaults |
| `renderer` | `"canvas" \| "svg"` | `"canvas"` | Rendering engine |
| `loading` | `boolean` | `false` | Show loading overlay |
| `autoResize` | `boolean` | `true` | Auto-resize on container change |
| `animateOnMount` | `boolean` | `true` | Animate on first render |
| `notMerge` | `boolean` | `false` | Replace option instead of merging |
| `lazyUpdate` | `boolean` | `false` | Defer chart update |
| `onEvents` | `Record<string, Function>` | -- | ECharts event handlers |
