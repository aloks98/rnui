---
title: Line Chart
description: Line charts with optional smoothing and multi-series support. Built on ECharts.
---

## Import

```tsx
import { LineChart } from '@e412/rnui-react'
```

## Usage

```tsx
<LineChart
  data={[
    { name: 'Jan', value: 120 },
    { name: 'Feb', value: 200 },
    { name: 'Mar', value: 150 },
    { name: 'Apr', value: 80 },
  ]}
  height={350}
/>
```

## Examples

### Smooth Lines

```tsx
<LineChart
  data={[
    { name: 'Mon', value: 820 },
    { name: 'Tue', value: 932 },
    { name: 'Wed', value: 901 },
    { name: 'Thu', value: 934 },
    { name: 'Fri', value: 1290 },
  ]}
  smooth
/>
```

### Multi-Series

```tsx
<LineChart
  categories={['Jan', 'Feb', 'Mar', 'Apr', 'May']}
  series={[
    { name: 'Revenue', data: [120, 200, 150, 80, 210] },
    { name: 'Expenses', data: [80, 120, 100, 90, 140], smooth: true },
  ]}
  showLegend
/>
```

### With Legend

```tsx
<LineChart
  data={[
    { name: 'Q1', value: 400 },
    { name: 'Q2', value: 300 },
    { name: 'Q3', value: 500 },
    { name: 'Q4', value: 450 },
  ]}
  showLegend
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `LineChartDataItem[]` | — | Array of `{ name, value }` objects |
| `categories` | `string[]` | — | Category labels (derived from `data` names if omitted) |
| `series` | `LineChartSeries[]` | — | Multi-series config: `{ name, data, smooth? }` |
| `smooth` | `boolean` | `false` | Use smooth curves |
| `showLegend` | `boolean` | `false` | Show the chart legend |
| `option` | `Partial<EChartsOption>` | — | Override any ECharts option |

Inherits all [EChart](/charts/) base props (`height`, `width`, `theme`, `preset`, `loading`, etc.).
