---
title: useIsMobile
description: A React hook that detects whether the current viewport is a mobile size (under 768px).
---

## Import

```tsx
import { useIsMobile } from '@e412/rnui-react'
```

## Usage

```tsx
import { useIsMobile } from '@e412/rnui-react'

function ResponsiveLayout() {
  const isMobile = useIsMobile()

  return (
    <div>
      {isMobile ? (
        <MobileNav />
      ) : (
        <DesktopNav />
      )}
    </div>
  )
}
```

## How It Works

The hook uses `window.matchMedia` to listen for viewport changes at the `768px` breakpoint. It returns `false` during SSR and on the initial render before the effect runs, then updates reactively as the viewport resizes.

## Return Value

| Value | Type | Description |
|-------|------|-------------|
| `isMobile` | `boolean` | `true` when viewport width is less than 768px |

## Examples

### Conditional Rendering

```tsx
const isMobile = useIsMobile()

return isMobile ? <Sheet>...</Sheet> : <Sidebar>...</Sidebar>
```

### Adjusting Props

```tsx
const isMobile = useIsMobile()

return <Dialog variant={isMobile ? 'fullscreen' : 'default'}>...</Dialog>
```
