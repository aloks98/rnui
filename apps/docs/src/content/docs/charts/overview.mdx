## Overview

The chart components wrap [Apache ECharts](https://echarts.apache.org/) with React bindings and automatic shadcn/ui theme integration. Charts automatically adapt to light and dark mode, use your design system's color tokens (`--chart-1` through `--chart-5`), and include sensible defaults via the built-in preset.

## Installation

ECharts is a peer dependency. Install it alongside the component library:

```bash
npm install echarts
```

Or with pnpm:

```bash
pnpm add echarts
```

## Import

```tsx
import { EChart } from '@e412/rnui-react'
```

## Base Component

The `EChart` component manages the full ECharts instance lifecycle -- initialization, option updates, theme switching, resize handling, and cleanup.

```tsx
import { EChart } from '@e412/rnui-react'

<EChart
  option={{
    xAxis: { type: 'category', data: ['Mon', 'Tue', 'Wed'] },
    yAxis: { type: 'value' },
    series: [{ type: 'bar', data: [120, 200, 150] }],
  }}
  height={350}
/>
```

## Theme Integration

Charts automatically detect whether your app is in light or dark mode by watching for the `.dark` class on `<html>`. The built-in `shadcn-light` and `shadcn-dark` themes map your CSS custom properties (`--chart-1` through `--chart-5`, `--background`, `--foreground`, `--border`, etc.) to ECharts theme values.

You can also pass a specific theme:

```tsx
<EChart option={option} theme="dark" />
<EChart option={option} theme="light" />
```

## Preset

The `preset` prop (enabled by default) applies minimal, opinionated defaults that match shadcn styling -- clean axis lines, subtle grid lines, proper spacing, and tooltip styling. Disable it for raw ECharts behavior:

```tsx
<EChart option={option} preset={false} />
```

## Configuration

Use the [chartConfig()](/charts/chart-config) builder API to programmatically create chart options with a fluent interface.

## Props (EChart)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `option` | `EChartsCoreOption` | — | ECharts option object |
| `width` | `number \| string` | `'100%'` | Chart width |
| `height` | `number \| string` | `350` | Chart height |
| `theme` | `string` | auto | Theme name (`'light'`, `'dark'`, or custom) |
| `preset` | `boolean` | `true` | Apply shadcn-style minimal defaults |
| `renderer` | `'canvas' \| 'svg'` | `'canvas'` | Rendering engine |
| `loading` | `boolean` | `false` | Show loading indicator |
| `loadingOption` | `LoadingOption` | — | Loading indicator options |
| `onEvents` | `Record<string, Function>` | — | ECharts event handlers |
| `autoResize` | `boolean` | `true` | Auto-resize on container change |
| `animateOnMount` | `boolean` | `true` | Run mount animation sequence |
| `animateOnMountDelayMs` | `number` | `16` | Delay before mount animation |
| `notMerge` | `boolean` | `false` | Replace option instead of merging |
| `lazyUpdate` | `boolean` | `false` | Lazy update mode |
| `className` | `string` | — | Additional CSS classes |
| `style` | `CSSProperties` | — | Inline styles |

## Ref

Access the ECharts instance via ref:

```tsx
import { useRef } from 'react'
import { EChart, type ChartRef } from '@e412/rnui-react'

function MyChart() {
  const chartRef = useRef<ChartRef>(null)

  return (
    <EChart
      ref={chartRef}
      option={option}
    />
  )
}

// chartRef.current.getEchartsInstance() — get raw ECharts instance
// chartRef.current.resize({ width, height }) — manually resize
```
