title: CodeBlock

---

import { CodeBlock, InlineCode } from '@e412/rnui-react';

CodeBlock renders syntax-highlighted code using Shiki. It supports multiple languages, line numbers, line highlighting, custom themes, and a built-in copy button.

## Import

```tsx
import { CodeBlock, InlineCode } from '@e412/rnui-react';
```

## Usage

<Preview>
  <CodeBlock
    code={`import { Button } from '@e412/rnui-react'

export function App() {
  return <Button>Click me</Button>
}`}
    language="tsx"
  />
</Preview>

## With Title

Add a title bar to label the code block, useful for filenames or descriptions.

<Preview>
  <CodeBlock
    code={`npm install @e412/rnui-react @e412/rnui-themes`}
    title="Terminal"
    language="bash"
  />
</Preview>

## With Line Numbers

Enable `showLineNumbers` to display line numbers alongside the code.

<Preview>
  <CodeBlock
    code={`import { Card, CardHeader, CardTitle, CardContent } from '@e412/rnui-react'

export function UserCard({ name, email }) {
  return (
    <Card className="w-[350px]">
      <CardHeader>
        <CardTitle>{name}</CardTitle>
      </CardHeader>
      <CardContent>
        <p>{email}</p>
      </CardContent>
    </Card>
  )
}`}
    language="tsx"
    showLineNumbers
  />
</Preview>

## With Highlighted Lines

Use `highlightLines` to draw attention to specific lines of code.

<Preview>
  <CodeBlock
    code={`import { useState } from 'react'
import { Button } from '@e412/rnui-react'

export function Counter() {
  const [count, setCount] = useState(0)

  return (
    <div>
      <p>Count: {count}</p>
      <Button onClick={() => setCount(c => c + 1)}>
        Increment
      </Button>
    </div>
  )
}`}
    language="tsx"
    showLineNumbers
    highlightLines={[5, 10, 11, 12]}
  />
</Preview>

## Without Copy Button

Set `showCopy` to `false` to hide the copy-to-clipboard button.

<Preview>
  <CodeBlock
    code={`// This block has no copy button
const x = 42`}
    language="ts"
    showCopy={false}
  />
</Preview>
