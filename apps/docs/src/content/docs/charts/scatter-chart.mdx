---
title: Scatter Chart
description: Scatter plot charts for visualizing data point distributions. Built on ECharts.
---

## Import

```tsx
import { ScatterChart } from '@e412/rnui-react'
```

## Usage

```tsx
<ScatterChart
  series={[
    {
      name: 'Sample',
      data: [
        [10, 8.04], [8, 6.95], [13, 7.58],
        [9, 8.81], [11, 8.33], [14, 9.96],
        [6, 7.24], [4, 4.26], [12, 10.84],
      ],
    },
  ]}
  height={350}
/>
```

## Examples

### Multi-Series

```tsx
<ScatterChart
  series={[
    {
      name: 'Group A',
      data: [
        [10, 8], [8, 7], [13, 8], [9, 9], [11, 8],
      ],
    },
    {
      name: 'Group B',
      data: [
        [6, 4], [4, 3], [12, 10], [14, 12], [7, 5],
      ],
    },
  ]}
  showLegend
/>
```

### Without Legend

```tsx
<ScatterChart
  series={[
    {
      name: 'Points',
      data: [
        [1, 2], [2, 3], [3, 5], [4, 4], [5, 6],
      ],
    },
  ]}
  showLegend={false}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `series` | `ScatterChartSeries[]` | — | Series data: `{ name, data: [x, y][] }` |
| `showLegend` | `boolean` | `true` | Show the chart legend |
| `option` | `Partial<EChartsOption>` | — | Override any ECharts option |

Inherits all [EChart](/charts/) base props (`height`, `width`, `theme`, `preset`, `loading`, etc.).
