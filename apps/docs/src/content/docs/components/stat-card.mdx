---
title: Stat Card
description: A dashboard metric card displaying a value, optional trend indicator, icon, and description.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { StatCard } from '@e412/rnui-react'
```

## Usage

```tsx
<StatCard title="Total Revenue" value="$45,231" />
```

## Examples

### With Trend

```tsx
<StatCard
  title="Subscribers"
  value="2,350"
  trend={{ value: 12.5, isPositive: true }}
/>
```

### With Negative Trend

```tsx
<StatCard
  title="Bounce Rate"
  value="24.3%"
  trend={{ value: 3.1, isPositive: false }}
/>
```

### With Icon

```tsx
import { DollarSignIcon } from 'lucide-react'

<StatCard
  title="Revenue"
  value="$12,450"
  icon={<DollarSignIcon />}
  trend={{ value: 8.2, isPositive: true }}
  description="Compared to last month"
/>
```

### Dashboard Grid

```tsx
import { UsersIcon, CreditCardIcon, ActivityIcon } from 'lucide-react'

<div className="grid gap-4 md:grid-cols-3">
  <StatCard
    title="Total Users"
    value="12,345"
    icon={<UsersIcon />}
    trend={{ value: 5.2, isPositive: true }}
  />
  <StatCard
    title="Revenue"
    value="$48,290"
    icon={<CreditCardIcon />}
    trend={{ value: 2.1, isPositive: false }}
  />
  <StatCard
    title="Active Now"
    value="573"
    icon={<ActivityIcon />}
    description="Real-time count"
  />
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | -- | **Required.** Metric label. |
| `value` | `string \| number` | -- | **Required.** The primary metric value. |
| `description` | `string` | -- | Supporting text below the value and trend. |
| `icon` | `ReactNode` | -- | Icon displayed on the right side. SVGs are sized to 20x20. |
| `trend` | `{ value: number; isPositive: boolean }` | -- | Trend indicator with percentage and direction arrow. |
| `className` | `string` | -- | Additional CSS classes. |
