---
title: ContextMenu
description: A menu triggered by right-clicking an element.
---

import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuLabel,
  ContextMenuSeparator,
  ContextMenuTrigger,
  ContextMenuCheckboxItem,
  ContextMenuRadioGroup,
  ContextMenuRadioItem,
  ContextMenuShortcut,
  ContextMenuSub,
  ContextMenuSubTrigger,
  ContextMenuSubContent,
} from '@e412/rnui-react';
import { Copy, Clipboard, Scissors, RotateCcw, Trash2, Share2, Link, Mail } from 'lucide-react';

The `ContextMenu` component displays a menu when the user right-clicks on a trigger area. It supports the same sub-components as `DropdownMenu` including labels, separators, keyboard shortcuts, and nested sub-menus.

## Import

```tsx
import {
  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuLabel,
  ContextMenuSeparator,
  ContextMenuTrigger,
  ContextMenuCheckboxItem,
  ContextMenuRadioGroup,
  ContextMenuRadioItem,
  ContextMenuShortcut,
  ContextMenuSub,
  ContextMenuSubTrigger,
  ContextMenuSubContent,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <ContextMenu>
    <ContextMenuTrigger className="flex h-[150px] w-[300px] items-center justify-center rounded-md border border-dashed text-sm">
      Right click here
    </ContextMenuTrigger>
    <ContextMenuContent className="w-64">
      <ContextMenuItem>Back</ContextMenuItem>
      <ContextMenuItem>Forward</ContextMenuItem>
      <ContextMenuItem>Reload</ContextMenuItem>
      <ContextMenuSeparator />
      <ContextMenuItem>View Page Source</ContextMenuItem>
      <ContextMenuItem>Inspect</ContextMenuItem>
    </ContextMenuContent>
  </ContextMenu>
</Preview>
