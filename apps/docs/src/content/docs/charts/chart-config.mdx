title: Chart Config

---

## Import

```tsx
import { chartConfig } from '@e412/rnui-react'
```

## Usage

The `chartConfig()` function returns a `ChartConfigBuilder` instance. Chain methods to define series, configure axes, and set options, then call `.build()` to produce the final ECharts option object.

```tsx
import { EChart, chartConfig } from '@e412/rnui-react'

const option = chartConfig()
  .bar({ data: [120, 200, 150], categories: ['A', 'B', 'C'] })
  .tooltip()
  .build()

<EChart option={option} />
```

## Examples

### Multi-Series Line Chart

```tsx
const option = chartConfig()
  .line({ data: [120, 200, 150], categories: ['Jan', 'Feb', 'Mar'], name: 'Revenue', smooth: true })
  .line({ data: [80, 120, 100], name: 'Expenses', smooth: true })
  .legend()
  .tooltip()
  .build()
```

### Donut Chart

```tsx
const option = chartConfig()
  .pie({
    data: [
      { name: 'Desktop', value: 65 },
      { name: 'Mobile', value: 30 },
      { name: 'Tablet', value: 5 },
    ],
    donut: true,
  })
  .tooltip('item')
  .legend()
  .build()
```

### Stacked Bar Chart

```tsx
const option = chartConfig()
  .bar({ data: [120, 200, 150], categories: ['Q1', 'Q2', 'Q3'], name: 'Product A', stack: 'total' })
  .bar({ data: [80, 120, 100], name: 'Product B', stack: 'total' })
  .legend()
  .tooltip()
  .build()
```

### Horizontal Bars

```tsx
const option = chartConfig()
  .bar({ data: [420, 310, 180], categories: ['React', 'Vue', 'Angular'], horizontal: true })
  .tooltip()
  .build()
```

### Line with Area Fill

```tsx
const option = chartConfig()
  .line({
    data: [120, 200, 150, 80, 210],
    categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May'],
    name: 'Revenue',
    smooth: true,
    area: true,
    areaOpacity: 0.2,
  })
  .tooltip()
  .build()
```

### Scatter Plot

```tsx
const option = chartConfig()
  .scatter({ data: [[10, 8], [8, 7], [13, 8], [9, 9]], name: 'Group A' })
  .scatter({ data: [[6, 4], [4, 3], [12, 10]], name: 'Group B' })
  .legend()
  .tooltip('item')
  .build()
```

### Radar Chart

```tsx
const option = chartConfig()
  .radar({
    indicators: [
      { name: 'Speed', max: 100 },
      { name: 'Reliability', max: 100 },
      { name: 'Cost', max: 100 },
    ],
    series: [
      { name: 'Product A', value: [90, 70, 60] },
      { name: 'Product B', value: [60, 85, 90] },
    ],
  })
  .legend()
  .tooltip('item')
  .build()
```

### Gauge Chart

```tsx
const option = chartConfig()
  .gauge({ value: 72, name: 'Score', min: 0, max: 100 })
  .build()
```

### With Data Zoom

```tsx
const option = chartConfig()
  .bar({ data: longDataArray, categories: longCategoryArray })
  .dataZoom(0, 50)
  .tooltip()
  .build()
```

### Custom Colors and Grid

```tsx
const option = chartConfig()
  .bar({ data: [120, 200, 150], categories: ['A', 'B', 'C'] })
  .colors(['#3b82f6', '#8b5cf6', '#ec4899'])
  .grid({ left: 20, right: 20, top: 30, bottom: 20 })
  .tooltip()
  .build()
```

### Raw Option Merge

For anything not covered by the helper methods, use `.raw()` to merge arbitrary ECharts options:

```tsx
const option = chartConfig()
  .bar({ data: [120, 200, 150], categories: ['A', 'B', 'C'] })
  .raw({ animation: false, backgroundColor: '#fafafa' })
  .build()
```

## API Reference

### Series Methods

| Method | Arguments | Description |
|--------|-----------|-------------|
| `.bar(config)` | `{ data, name?, categories?, stack?, horizontal?, borderRadius?, barWidth? }` | Add a bar series |
| `.line(config)` | `{ data, name?, categories?, smooth?, area?, areaOpacity? }` | Add a line series |
| `.pie(config)` | `{ data, donut?, innerRadius?, outerRadius? }` | Add a pie/donut series |
| `.scatter(config)` | `{ data, name?, symbolSize? }` | Add a scatter series |
| `.radar(config)` | `{ indicators, series }` | Add a radar chart |
| `.gauge(config)` | `{ value, name?, min?, max? }` | Add a gauge chart |

### Configuration Methods

| Method | Arguments | Description |
|--------|-----------|-------------|
| `.title(text, subtitle?)` | `string, string?` | Set chart title (ECharts-rendered) |
| `.legend(position?)` | `'top' \| 'bottom' \| 'left' \| 'right'` | Enable legend (default: `'bottom'`) |
| `.tooltip(trigger?, formatter?)` | `'axis' \| 'item'`, `string \| Function` | Enable tooltip (default trigger: `'axis'`) |
| `.dataZoom(start?, end?)` | `number, number` | Enable data zoom slider (default: `0`-`100`) |
| `.grid(margins)` | `{ left?, right?, top?, bottom? }` | Set custom grid margins |
| `.colors(palette)` | `string[]` | Set custom color palette |
| `.raw(opts)` | `Partial<EChartsOption>` | Merge arbitrary ECharts options |
| `.build()` | — | Build the final `EChartsOption` object |
