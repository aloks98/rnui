---
title: ButtonGroup
description: Groups related buttons together with shared borders and spacing.
---

import { ButtonGroup, ButtonGroupSeparator, Button } from '@e412/rnui-react';
import { Plus, Minus, ArrowLeft, ArrowRight, Search } from 'lucide-react';

The `ButtonGroup` component visually groups related buttons together, merging their borders for a cohesive look. Supports horizontal and vertical orientations.

## Import

```tsx
import { ButtonGroup, ButtonGroupSeparator, Button } from '@e412/rnui-react';
```

## Usage

<Preview>
  <ButtonGroup>
    <Button variant="outline">Archive</Button>
    <Button variant="outline">Report</Button>
    <Button variant="outline">Snooze</Button>
  </ButtonGroup>
</Preview>

```tsx
<ButtonGroup>
  <Button variant="outline">Archive</Button>
  <Button variant="outline">Report</Button>
  <Button variant="outline">Snooze</Button>
</ButtonGroup>
```
