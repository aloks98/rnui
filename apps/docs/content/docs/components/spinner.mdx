---
title: Spinner
description: A loading indicator that signals an ongoing process.
---

import { Spinner, Button } from '@e412/rnui-react';

The `Spinner` component provides a visual loading indicator. Use it to communicate that content or an action is in progress.

## Import

```tsx
import { Spinner } from '@e412/rnui-react';
```

## Usage

<Preview>
  <Spinner />
</Preview>

```tsx
<Spinner />
```

## Sizes

Control the size using Tailwind's `size-*` utility classes.

<Preview>
  <div className="flex gap-4 items-center">
    <Spinner className="size-3" />
    <Spinner />
    <Spinner className="size-6" />
    <Spinner className="size-8" />
  </div>
</Preview>

```tsx
<div className="flex gap-4 items-center">
    <Spinner className="size-3" />
    <Spinner />
    <Spinner className="size-6" />
    <Spinner className="size-8" />
  </div>
```
