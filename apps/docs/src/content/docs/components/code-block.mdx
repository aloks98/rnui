---
title: CodeBlock
description: Syntax-highlighted code block powered by Shiki with line numbers, line highlighting, dual themes, and a built-in copy button.
---

## Installation

```bash
npx @e412/rnui-cli add code-block
```

## Import

```tsx
import { CodeBlock, InlineCode } from '@e412/rnui-react'
```

## Usage

```tsx
<CodeBlock
  code={`const greeting = "Hello, world!";
console.log(greeting);`}
  language="typescript"
/>
```

## Examples

### Basic Code Block

```tsx
<CodeBlock
  code={`function add(a: number, b: number) {
  return a + b;
}`}
  language="typescript"
/>
```

### With Title

```tsx
<CodeBlock
  title="utils.ts"
  code={`export function cn(...classes: string[]) {
  return classes.filter(Boolean).join(" ");
}`}
  language="typescript"
/>
```

### Line Numbers

```tsx
<CodeBlock
  code={`import React from 'react'

function App() {
  return <h1>Hello</h1>
}

export default App`}
  language="tsx"
  showLineNumbers
/>
```

### Highlight Lines

Draw attention to specific lines with `highlightLines`.

```tsx
<CodeBlock
  code={`function fetchUser(id: string) {
  const response = await fetch(\`/api/users/\${id}\`);
  const user = await response.json();
  return user;
}`}
  language="typescript"
  showLineNumbers
  highlightLines={[2, 3]}
/>
```

### Custom Themes

Override the default light/dark theme pair.

```tsx
<CodeBlock
  code={`console.log("custom theme")`}
  language="javascript"
  themes={{
    light: "vitesse-light",
    dark: "vitesse-dark",
  }}
/>
```

### Custom Font Size

```tsx
<CodeBlock
  code={`const x = 42;`}
  language="typescript"
  fontSize="16px"
/>
```

### Without Copy Button

```tsx
<CodeBlock
  code={`echo "no copy button"`}
  language="bash"
  showCopy={false}
/>
```

### Inline Code

Use `InlineCode` for short code snippets within text.

```tsx
<p>
  Run <InlineCode>npm install</InlineCode> to install dependencies.
</p>
```

## Props

### CodeBlock

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `code` | `string` | — | The source code to display |
| `language` | `string` | `"text"` | Programming language for syntax highlighting |
| `showLineNumbers` | `boolean` | `false` | Show line numbers in the gutter |
| `showCopy` | `boolean` | `true` | Show the copy-to-clipboard button |
| `title` | `string` | — | File name or title shown in the header |
| `highlightLines` | `number[]` | `[]` | Line numbers to highlight |
| `themes` | `{ light: BundledTheme; dark: BundledTheme }` | `{ light: "github-light-default", dark: "github-dark-default" }` | Shiki theme pair for light and dark mode |
| `fontSize` | `string` | `"13px"` | CSS font size for the code |
| `className` | `string` | — | Additional CSS classes |

### InlineCode

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Inline code text |
| `className` | `string` | — | Additional CSS classes |
