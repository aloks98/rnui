title: Card
description: A container for grouping related content and actions.
---

import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter, Button } from '@e412/rnui-react';

The `Card` component provides a flexible container for displaying grouped content. It supports a header with title and description, a content area, and a footer for actions.

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from '@e412/rnui-react';
```

## Usage

<Preview code={`<Card className="w-[350px]">
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card description goes here.</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card content goes here.</p>
  </CardContent>
  <CardFooter>
    <Button>Action</Button>
  </CardFooter>
</Card>`}>
  <Card className="w-[350px]">
    <CardHeader>
      <CardTitle>Card Title</CardTitle>
      <CardDescription>Card description goes here.</CardDescription>
    </CardHeader>
    <CardContent>
      <p>Card content goes here.</p>
    </CardContent>
    <CardFooter>
      <Button>Action</Button>
    </CardFooter>
  </Card>
</Preview>

## Variants

### Simple

A card with content only, no header or footer.

<Preview code={`<Card className="w-[350px]">
  <CardContent className="pt-6">
    <p>A simple card with just content.</p>
  </CardContent>
</Card>`}>
  <Card className="w-[350px]">
    <CardContent className="pt-6">
      <p>A simple card with just content.</p>
    </CardContent>
  </Card>
</Preview>

### Header Only

A card with only a header section.

<Preview code={`<Card className="w-[350px]">
  <CardHeader>
    <CardTitle>Notifications</CardTitle>
    <CardDescription>You have 3 unread messages.</CardDescription>
  </CardHeader>
</Card>`}>
  <Card className="w-[350px]">
    <CardHeader>
      <CardTitle>Notifications</CardTitle>
      <CardDescription>You have 3 unread messages.</CardDescription>
    </CardHeader>
  </Card>
</Preview>
