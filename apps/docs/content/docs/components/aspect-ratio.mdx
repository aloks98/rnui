---
title: AspectRatio
description: Displays content within a desired ratio.
---

import { AspectRatio } from '@e412/rnui-react';

The `AspectRatio` component enforces a consistent width-to-height ratio on its child content. Use it to maintain proportional dimensions for images, videos, maps, or any embedded media.

## Import

```tsx
import { AspectRatio } from '@e412/rnui-react';
```

## Usage

<Preview code={`<div className="w-[450px]">
  <AspectRatio ratio={16 / 9}>
    <div className="flex h-full w-full items-center justify-center rounded-md bg-muted text-muted-foreground">
      16:9 Aspect Ratio
    </div>
  </AspectRatio>
</div>`}>
  <div className="w-[450px]">
    <AspectRatio ratio={16 / 9}>
      <div className="flex h-full w-full items-center justify-center rounded-md bg-muted text-muted-foreground">
        16:9 Aspect Ratio
      </div>
    </AspectRatio>
  </div>
</Preview>
