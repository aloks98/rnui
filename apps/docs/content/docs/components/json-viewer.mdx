---
title: JsonViewer
description: An interactive viewer for exploring JSON data with collapsible nodes.
---

import { JsonViewer } from '@e412/rnui-react';

The `JsonViewer` component renders JSON data in an expandable tree with syntax highlighting. It supports controlling default expansion depth, line numbers, colored indent guides, and array truncation.

## Import

```tsx
import { JsonViewer } from '@e412/rnui-react';
```

## Usage

<Preview code={`<JsonViewer
    title="User Data"
    data={{
      name: "John Doe",
      age: 30,
      active: true,
      email: "john@example.com",
      tags: ["developer", "designer"],
    }}
  />`}>
  <JsonViewer
    title="User Data"
    data={{
      name: "John Doe",
      age: 30,
      active: true,
      email: "john@example.com",
      tags: ["developer", "designer"],
    }}
  />
</Preview>

## Expanded by Default

Pass `defaultExpanded={true}` to expand all nodes on mount.

<Preview code={`<JsonViewer
    title="Fully Expanded"
    defaultExpanded={true}
    data={{
      name: "John Doe",
      age: 30,
      address: {
        street: "123 Main St",
        city: "Springfield",
        country: "US",
      },
      tags: ["developer", "designer", "writer"],
    }}
  />`}>
  <JsonViewer
    title="Fully Expanded"
    defaultExpanded={true}
    data={{
      name: "John Doe",
      age: 30,
      address: {
        street: "123 Main St",
        city: "Springfield",
        country: "US",
      },
      tags: ["developer", "designer", "writer"],
    }}
  />
</Preview>

## Expand to Depth

Control the default expansion depth with a number value.

<Preview code={`<JsonViewer
    title="Expanded to Level 1"
    defaultExpanded={1}
    data={{
      name: "John Doe",
      address: {
        street: "123 Main St",
        city: "Springfield",
      },
      projects: [
        { id: 1, name: "Project Alpha", status: "active" },
        { id: 2, name: "Project Beta", status: "completed" },
      ],
    }}
  />`}>
  <JsonViewer
    title="Expanded to Level 1"
    defaultExpanded={1}
    data={{
      name: "John Doe",
      address: {
        street: "123 Main St",
        city: "Springfield",
      },
      projects: [
        { id: 1, name: "Project Alpha", status: "active" },
        { id: 2, name: "Project Beta", status: "completed" },
      ],
    }}
  />
</Preview>
