---
title: Resizable
description: Draggable panels that can be resized within a group.
---

import { ResizablePanelGroup, ResizablePanel, ResizableHandle } from '@e412/rnui-react';

The `Resizable` components let you create draggable, resizable panel layouts. Use `ResizablePanelGroup` to define the direction, `ResizablePanel` for each section, and `ResizableHandle` for the drag handle between panels.

## Import

```tsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <ResizablePanelGroup direction="horizontal" className="max-w-md rounded-lg border">
    <ResizablePanel defaultSize={50}>
      <div className="flex h-[200px] items-center justify-center p-6">
        <span className="font-semibold">One</span>
      </div>
    </ResizablePanel>
    <ResizableHandle withHandle />
    <ResizablePanel defaultSize={50}>
      <div className="flex h-[200px] items-center justify-center p-6">
        <span className="font-semibold">Two</span>
      </div>
    </ResizablePanel>
  </ResizablePanelGroup>
</Preview>
