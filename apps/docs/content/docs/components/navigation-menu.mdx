---
title: NavigationMenu
description: A horizontal navigation bar with dropdown content panels.
---

import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
} from '@e412/rnui-react';

The `NavigationMenu` component provides a horizontal navigation bar with rich dropdown content. It is well suited for site-wide navigation with categorized links and descriptive panels.

## Import

```tsx
import {
  NavigationMenu,
  NavigationMenuContent,
  NavigationMenuItem,
  NavigationMenuLink,
  NavigationMenuList,
  NavigationMenuTrigger,
} from '@e412/rnui-react';
```
