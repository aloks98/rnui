---
title: Badge
description: A small label used to highlight status, category, or count.
---

import { Badge } from '@e412/rnui-react';

The `Badge` component renders a compact label. It supports solid, light, and outline variant families, five sizes, and two border-radius options.

## Import

```tsx
import { Badge } from '@e412/rnui-react';
```

## Usage

<Preview>
  <Badge>Badge</Badge>
</Preview>

```tsx
<Badge>Badge</Badge>
```
