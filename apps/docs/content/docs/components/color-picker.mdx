---
title: ColorPicker
description: A color picker with saturation/brightness area, hue slider, alpha control, and hex/rgb/hsl format switching.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { ColorPicker } from '@e412/rnui-react'
```

## Usage

```tsx
const [color, setColor] = React.useState('#3b82f6')

<ColorPicker value={color} onChange={setColor} />
```

## Examples

### Default

```tsx
<ColorPicker value="#3b82f6" onChange={console.log} />
```

### With RGB Format

```tsx
<ColorPicker value="#3b82f6" defaultFormat="rgb" onChange={console.log} />
```

### With HSL Format

```tsx
<ColorPicker value="#3b82f6" defaultFormat="hsl" onChange={console.log} />
```

### With Alpha Slider

```tsx
<ColorPicker value="#3b82f6" showAlpha onChange={console.log} />
```

### With Preset Colors

```tsx
<ColorPicker
  value="#3b82f6"
  presets={[
    '#ef4444', '#f97316', '#eab308', '#22c55e',
    '#3b82f6', '#8b5cf6', '#ec4899', '#000000',
  ]}
  onChange={console.log}
/>
```

### Full Featured

```tsx
<ColorPicker
  value="#8b5cf6"
  defaultFormat="hex"
  showAlpha
  presets={[
    '#ef4444', '#f97316', '#eab308', '#22c55e',
    '#3b82f6', '#8b5cf6', '#ec4899', '#6b7280',
  ]}
  onChange={(color) => console.log('Selected:', color)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | `"#3b82f6"` | Current color value (any CSS color string) |
| `onChange` | `(value: string) => void` | — | Callback when color changes |
| `defaultFormat` | `"hex" \| "rgb" \| "hsl"` | `"hex"` | Initial display format |
| `presets` | `string[]` | — | Preset color swatches |
| `showAlpha` | `boolean` | `false` | Show alpha/opacity slider |
| `className` | `string` | — | Additional CSS classes for the trigger button |
