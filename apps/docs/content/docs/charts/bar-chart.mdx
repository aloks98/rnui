import { BarChart } from '@e412/rnui-react';

The `BarChart` component renders vertical or horizontal bar charts. It accepts a simple `data` array for single-series charts, or `categories` and `series` for multi-series and stacked layouts.

## Import

```tsx
import { BarChart } from '@e412/rnui-react';
```

## Negative Values

Bars automatically handle negative values. Use the `option` prop to customize colors per value.

<Preview code={`<div className="w-full max-w-lg">
    <BarChart
      data={[
        { name: 'Jan', value: 4200 },
        { name: 'Feb', value: -1800 },
        { name: 'Mar', value: 6500 },
        { name: 'Apr', value: -2400 },
        { name: 'May', value: 3100 },
        { name: 'Jun', value: -3200 },
      ]}
      height={350}
      option={{
        series: [
          {
            type: 'bar',
            data: [4200, -1800, 6500, -2400, 3100, -3200].map((v) => ({
              value: v,
              itemStyle: {
                borderRadius: v >= 0 ? [4, 4, 0, 0] : [0, 0, 4, 4],
                color: v >= 0 ? '#22c55e' : '#ef4444',
              },
            })),
            barMaxWidth: 40,
          },
        ],
        xAxis: { type: 'category', data: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'], axisTick: { show: false } },
        yAxis: { type: 'value', splitLine: { lineStyle: { type: 'dashed', opacity: 0.5 } } },
        grid: { containLabel: true, left: 16, right: 16, top: 24, bottom: 8 },
      }}
    />
  </div>`}>
  <div className="w-full max-w-lg">
    <BarChart
      data={[
        { name: 'Jan', value: 4200 },
        { name: 'Feb', value: -1800 },
        { name: 'Mar', value: 6500 },
        { name: 'Apr', value: -2400 },
        { name: 'May', value: 3100 },
        { name: 'Jun', value: -3200 },
      ]}
      height={350}
      option={{
        series: [
          {
            type: 'bar',
            data: [4200, -1800, 6500, -2400, 3100, -3200].map((v) => ({
              value: v,
              itemStyle: {

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `{ name: string; value: number }[]` | required | Data items for single-series charts |
| `categories` | `string[]` | -- | X-axis category labels (for multi-series) |
| `series` | `object[]` | -- | ECharts series config (for multi-series/stacked) |
| `horizontal` | `boolean` | `false` | Render bars horizontally |
| `showLegend` | `boolean` | `false` | Show chart legend |
| `height` | `number` | `350` | Chart height in pixels |
| `loading` | `boolean` | `false` | Show loading state |
| `option` | `object` | -- | Override or extend the ECharts option |
