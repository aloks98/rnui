import { ScatterChart } from '@e412/rnui-react';

The `ScatterChart` component renders scatter plots. It accepts one or more `series`, each containing an array of `[x, y]` data points.

## Import

```tsx
import { ScatterChart } from '@e412/rnui-react';
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `series` | `{ name: string; data: [number, number][] }[]` | required | Data series |
| `showLegend` | `boolean` | `false` | Show chart legend |
| `height` | `number` | `350` | Chart height in pixels |
| `loading` | `boolean` | `false` | Show loading state |
| `option` | `object` | -- | Override or extend the ECharts option |
