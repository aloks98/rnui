---
title: JSON Viewer
description: A collapsible JSON tree viewer with line numbers, color-coded indentation, smart value rendering, and expand/collapse controls.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { JsonViewer } from '@e412/rnui-react'
```

## Usage

```tsx
const data = {
  name: "John Doe",
  age: 30,
  active: true,
  address: {
    city: "New York",
    zip: "10001",
  },
}

<JsonViewer data={data} />
```

## Examples

### With Title

```tsx
<JsonViewer data={{ status: "ok", code: 200 }} title="API Response" />
```

### Color-coded Indentation

```tsx
<JsonViewer
  data={{ users: [{ name: "Alice" }, { name: "Bob" }] }}
  showColorIndent
/>
```

### Without Line Numbers

```tsx
<JsonViewer data={{ key: "value" }} showLineNumbers={false} />
```

### Default Expanded to a Depth

```tsx
<JsonViewer
  data={{ a: { b: { c: { d: "deep" } } } }}
  defaultExpanded={2}
/>
```

### Double-click to Collapse

```tsx
<JsonViewer data={{ items: [1, 2, 3] }} collapseOn="doubleClick" />
```

### Custom Truncation

```tsx
<JsonViewer
  data={{ list: Array.from({ length: 50 }, (_, i) => i) }}
  truncation={{ enabled: true, itemsPerArray: 10 }}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `Record<string, any>` | -- | **Required.** The JSON data object to display. |
| `title` | `string` | -- | Optional title shown in the viewer header. |
| `showLineNumbers` | `boolean` | `true` | Show line numbers on the left side. |
| `showColorIndent` | `boolean` | `false` | Use color-coded indent guides instead of plain borders. |
| `collapseOn` | `"click" \| "doubleClick"` | `"click"` | Interaction to toggle node expansion. |
| `defaultExpanded` | `boolean \| number` | `false` | Initial expansion state. `true` expands all, a number expands to that depth. |
| `truncation` | `{ enabled?: boolean; itemsPerArray?: number }` | `{ enabled: true, itemsPerArray: 5 }` | Controls array truncation behavior. |
| `className` | `string` | -- | Additional CSS classes. |
