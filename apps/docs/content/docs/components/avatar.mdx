---
title: Avatar
description: A circular image or fallback representing a user or entity.
---

import { Avatar, AvatarImage, AvatarFallback, AvatarBadge, AvatarGroup, AvatarGroupCount } from '@e412/rnui-react';

The `Avatar` component displays a user image with a fallback for when the image is unavailable. It supports three sizes, an online-status badge, and grouping.

## Import

```tsx
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarBadge,
  AvatarGroup,
  AvatarGroupCount,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <Avatar>
    <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
    <AvatarFallback>CN</AvatarFallback>
  </Avatar>
</Preview>

## Fallback

When no image is provided, the fallback content (typically initials) is shown.

<Preview>
  <Avatar>
    <AvatarFallback>AB</AvatarFallback>
  </Avatar>
</Preview>

## With Badge

Add `AvatarBadge` to indicate online status or notifications.

<Preview>
  <div className="flex gap-4 items-center">
    <Avatar size="sm">
      <AvatarFallback>SM</AvatarFallback>
      <AvatarBadge />
    </Avatar>
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      <AvatarBadge />
    </Avatar>
    <Avatar size="lg">
      <AvatarFallback>LG</AvatarFallback>
      <AvatarBadge />
    </Avatar>
  </div>
</Preview>

## Avatar Group

Stack multiple avatars together using `AvatarGroup` and show an overflow count with `AvatarGroupCount`.

<Preview>
  <AvatarGroup>
    <Avatar>
      <AvatarFallback>A</AvatarFallback>
    </Avatar>
    <Avatar>
      <AvatarFallback>B</AvatarFallback>
    </Avatar>
    <Avatar>
      <AvatarFallback>C</AvatarFallback>
    </Avatar>
    <AvatarGroupCount>+3</AvatarGroupCount>
  </AvatarGroup>
</Preview>
