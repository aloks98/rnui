title: NumberField
description: A numeric input with increment and decrement controls.
---

import { NumberField, NumberFieldGroup, NumberFieldDecrement, NumberFieldIncrement, NumberFieldInput, NumberFieldScrubArea, Label } from '@e412/rnui-react';

The `NumberField` component provides a structured numeric input with built-in increment/decrement buttons, keyboard support, min/max constraints, and configurable step values. It is composed of several sub-components for flexible layout.

## Import

```tsx
import {
  NumberField,
  NumberFieldGroup,
  NumberFieldDecrement,
  NumberFieldIncrement,
  NumberFieldInput,
  NumberFieldScrubArea,
  Label,
} from '@e412/rnui-react';
```

## Usage

<Preview>
  <NumberField defaultValue={5} className="w-36">
    <NumberFieldGroup>
      <NumberFieldDecrement />
      <NumberFieldInput />
      <NumberFieldIncrement />
    </NumberFieldGroup>
  </NumberField>
</Preview>

