---
title: CopyButton
description: A button that copies a text value to the clipboard with visual feedback.
---

import { CopyButton } from '@e412/rnui-react';

CopyButton copies a given text value to the user's clipboard when clicked. It provides visual feedback by briefly showing a checkmark icon after a successful copy.

## Import

```tsx
import { CopyButton } from '@e412/rnui-react';
```

## Usage

<Preview>
  <CopyButton value="npm install @e412/rnui-react" />
</Preview>

```tsx
<CopyButton value="npm install @e412/rnui-react" />
```

## With Inline Content

Pair CopyButton alongside text or code snippets to let users quickly copy values.

<Preview>
  <div className="flex items-center gap-2">
    <code className="rounded bg-muted px-2 py-1 text-sm font-mono">npm install @e412/rnui-react</code>
    <CopyButton value="npm install @e412/rnui-react" />
  </div>
</Preview>

<div className="flex items-center gap-2">
