---
title: Bar Chart
description: Vertical and horizontal bar charts with optional stacking. Built on ECharts.
---

## Import

```tsx
import { BarChart } from '@e412/rnui-react'
```

## Usage

```tsx
<BarChart
  data={[
    { name: 'Jan', value: 120 },
    { name: 'Feb', value: 200 },
    { name: 'Mar', value: 150 },
    { name: 'Apr', value: 80 },
  ]}
  height={350}
/>
```

## Examples

### Horizontal

```tsx
<BarChart
  data={[
    { name: 'React', value: 420 },
    { name: 'Vue', value: 310 },
    { name: 'Angular', value: 180 },
  ]}
  horizontal
/>
```

### Stacked

```tsx
<BarChart
  data={[
    { name: 'Q1', value: 120 },
    { name: 'Q2', value: 200 },
    { name: 'Q3', value: 150 },
  ]}
  stacked
  showLegend
/>
```

### Custom Series

Pass custom ECharts series for multi-series charts:

```tsx
<BarChart
  data={[]}
  categories={['Jan', 'Feb', 'Mar']}
  series={[
    { type: 'bar', name: 'Revenue', data: [120, 200, 150], stack: 'total' },
    { type: 'bar', name: 'Expenses', data: [80, 120, 100], stack: 'total' },
  ]}
  showLegend
/>
```

### With Legend

```tsx
<BarChart
  data={[
    { name: 'A', value: 40 },
    { name: 'B', value: 60 },
    { name: 'C', value: 35 },
  ]}
  showLegend
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `BarChartDataItem[]` | — | Array of `{ name, value }` objects |
| `categories` | `string[]` | — | Category labels (derived from `data` names if omitted) |
| `series` | `EChartsOption['series']` | — | Custom ECharts series config |
| `horizontal` | `boolean` | `false` | Render bars horizontally |
| `stacked` | `boolean` | `false` | Stack bars |
| `showLegend` | `boolean` | `false` | Show the chart legend |
| `option` | `Partial<EChartsOption>` | — | Override any ECharts option |

Inherits all [EChart](/charts/) base props (`height`, `width`, `theme`, `preset`, `loading`, etc.).
