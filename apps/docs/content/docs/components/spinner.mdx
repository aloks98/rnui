---
title: Spinner
description: A simple animated loading spinner indicator.
---

## Installation

```bash
npm install @e412/rnui-react @e412/rnui-themes
```

## Import

```tsx
import { Spinner } from '@e412/rnui-react'
```

## Usage

```tsx
<Spinner />
```

## Examples

### Custom Size

```tsx
<Spinner className="size-8" />
```

### With Label

```tsx
<div className="flex items-center gap-2">
  <Spinner />
  <span className="text-sm text-muted-foreground">Loading...</span>
</div>
```

### Inside a Button

```tsx
import { Button } from '@e412/rnui-react'

<Button disabled>
  <Spinner />
  Please wait
</Button>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes. Default size is `size-4`. |

All standard SVG element props are also supported.
