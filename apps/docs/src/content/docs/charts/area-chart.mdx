---
title: Area Chart
description: Area charts with gradient fills, stacking, and multi-series support. Built on ECharts.
---

## Import

```tsx
import { AreaChart } from '@e412/rnui-react'
```

## Usage

```tsx
<AreaChart
  categories={['Jan', 'Feb', 'Mar', 'Apr', 'May']}
  series={[
    { name: 'Revenue', data: [120, 200, 150, 80, 210] },
  ]}
  height={350}
/>
```

## Examples

### Multi-Series with Gradient

```tsx
<AreaChart
  categories={['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']}
  series={[
    { name: 'Desktop', data: [120, 200, 150, 80, 210, 180] },
    { name: 'Mobile', data: [80, 120, 100, 90, 140, 160] },
  ]}
  gradient
  showLegend
/>
```

### Stacked

```tsx
<AreaChart
  categories={['Mon', 'Tue', 'Wed', 'Thu', 'Fri']}
  series={[
    { name: 'Email', data: [120, 132, 101, 134, 90] },
    { name: 'Social', data: [220, 182, 191, 234, 290] },
    { name: 'Direct', data: [150, 232, 201, 154, 190] },
  ]}
  stacked
  showLegend
/>
```

### Without Gradient

```tsx
<AreaChart
  categories={['Q1', 'Q2', 'Q3', 'Q4']}
  series={[
    { name: 'Sales', data: [400, 300, 500, 450] },
  ]}
  gradient={false}
/>
```

### Smooth Curves

```tsx
<AreaChart
  categories={['Jan', 'Feb', 'Mar', 'Apr']}
  series={[
    { name: 'Users', data: [100, 250, 180, 320], smooth: true },
  ]}
  smooth
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `categories` | `string[]` | — | Category labels for the x-axis |
| `series` | `AreaChartSeries[]` | — | Series config: `{ name, data, smooth?, color? }` |
| `smooth` | `boolean` | `true` | Use smooth curves |
| `stacked` | `boolean` | `false` | Stack series |
| `showLegend` | `boolean` | `true` | Show the chart legend |
| `gradient` | `boolean` | `true` | Use gradient fill from series color to transparent |
| `option` | `Partial<EChartsOption>` | — | Override any ECharts option |

Inherits all [EChart](/charts/) base props (`height`, `width`, `theme`, `preset`, `loading`, etc.).
