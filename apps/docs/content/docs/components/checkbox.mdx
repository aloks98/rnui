---
title: Checkbox
description: A control that allows selecting one or more options.
---

import { Checkbox, Label } from '@e412/rnui-react';

The `Checkbox` component renders a styled checkbox input. Pair it with `Label` for accessible, clickable labels.

## Import

```tsx
import { Checkbox } from '@e412/rnui-react';
```

## Usage

<Preview>
  <div className="flex items-center gap-2">
    <Checkbox id="terms" />
    <Label htmlFor="terms">Accept terms and conditions</Label>
  </div>
</Preview>

```tsx
<div className="flex items-center gap-2">
  <Checkbox id="terms" />
  <Label htmlFor="terms">Accept terms and conditions</Label>
</div>
```
